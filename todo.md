- 'cross 신호 종목(signal-pulse의 vision + kis api 방식 중 적극매수 종목) + theme-analysis 대장주'를 대상으로 원격에 수집되어 있는 분봉 데이터를 활용해서 아래 계산을 수행하고 연구를 실시
- 변수 a, b, c 설정
- a는 익절 한계값, b는 손절 한계값, c는 급락 시 손절 기준값
- 시나리오
  - 매집 후 해당 종목이 a값(ex. 10%)에 도달할때까지 대기하다가, 도달 시 익절 실시
  - 매집 후 해당 종목의 수익률이 양수일 때는 a값에 도달할때까지 계속 대기
  - 매집 후 해당 종목의 수익률이 양수라고 하더라도 급락하여 c값만큼 급락했으면 해당 시점에 익절 or 손절
  - 기본 손절 한계값은 b로 설정
- 해당 시나리오대로 모의투자를 진행했을 때, 최적의 a,b,c 값을 도출(ex. a: 10, b: 3 c: 5)
