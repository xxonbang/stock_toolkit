## 아이디어 16: 공매도 역발상 시그널

### 개요
공매도 비율이 급증한 종목 중에서 **펀더멘탈 대비 과도하게 공격받은 종목**을 탐지하여 역발상 매수 기회를 포착하는 시스템. 공매도 세력의 숏커버링(short squeeze) 가능성이 높은 종목을 선별하고, 숏커버 시작 징후를 실시간으로 감지한다.

### 핵심 가치
- **분석 깊이**: 공매도 비율 단일 지표가 아닌 PER/PBR/수급/테마 종합 판단
- **실시간성**: 숏커버링 시작 징후(공매도 감소 + 거래량 증가) 즉시 포착
- **편의성**: "과매도된 우량주" 리스트를 자동으로 관리

### 작동 방식 (핵심 분석)

#### 1단계: 공매도 과열 종목 필터링
```
필터 조건:
1. 공매도 비율 > 5% (평소 대비 상위)
2. 공매도 비율 5일 연속 증가
3. BUT PER < 업종 평균 또는 외국인 순매수 전환

예시: 에코프로비엠 (247540)
• 공매도 비율: 6.8% (20일 전 3.2% → 113% 증가)
• PER: 18.5 (2차전지 업종 평균 25.3 대비 할인)
• 외국인: 3일 전 매도 → 어제부터 순매수 전환
→ "공매도 과열 + 펀더멘탈 양호" = 숏스퀴즈 후보
```

#### 2단계: 숏커버링 시작 징후 감지
```
숏커버 신호 판정 기준:
① 공매도 비율 3일 연속 감소 (정리 시작)
② 거래량 20일 평균의 200%+ 동반 (커버링 물량)
③ 가격 하방 지지선 확인 후 반등 시작
④ 테마 전체 반등과 동시 발생 (섹터 모멘텀)

4가지 중 3가지 이상 충족 → 숏커버 알림 발생
```

#### 3단계: 과거 숏스퀴즈 사례 통계
```
과거 90일 숏스퀴즈 패턴 발생 후 수익률:
• 표본: 23건 (공매도 비율 5%+ → 3일 감소 + 거래량 급증)
• D+3 평균 수익: +4.8% (승률 78%)
• D+5 평균 수익: +7.2% (승률 74%)
• D+10 평균 수익: +5.1% (승률 61%) ← 일부 재하락
→ 최적 보유 기간: 3~5일
```

### 텔레그램 알림 예시
```
🔄 [숏커버 시그널] 숏스퀴즈 후보 감지
━━━━━━━━━━━━━━━━━

📌 에코프로비엠 (247540)
• 공매도 비율: 6.8% → 5.9% (3일 연속 감소 ↓)
• 거래량: 20일 평균의 280% (커버링 물량 추정)
• 가격: 128,500원 (+2.8%) — 60일 지지선 반등
• PER 18.5 (업종 평균 25.3 — 28% 할인)

📊 과거 유사 패턴 (공매도 6%+ → 감소 + 거래량 급증)
• 23건 중 D+5 평균 +7.2% (승률 74%)
• 최대: +18.5% (한미반도체 2025-12)
• 최소: -4.2% (카카오 2026-01)

💡 판단: 공매도 과열 해소 + 외국인 전환 + 업종 저평가
   숏커버링 초기 단계로 추정. 3~5일 보유 전략 검토.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 공매도 비율 데이터 | theme_analysis → 종목별 공매도 비율 (일별 수집) |
| PER/PBR/시가총액 | theme_analysis → 펀더멘탈 비교 기준 |
| 투자자별 매매동향 | theme_analysis → 외국인 순매수 전환 감지 |
| 거래량 프로파일 | theme_analysis → 정상 거래량 대비 급증 판단 |
| 종목 시세 히스토리 (90일) | theme_analysis → 지지선/저항선 산출 |
| 테마 분류 | theme_analysis → 동일 섹터 비교 |
| 매매 신호 | signal_analysis → 기술적 분석 교차 검증 |

### 기술 구현
- **공매도 추적**: 기존 theme_analysis 공매도 비율 일별 데이터 활용, 변화율 계산
- **필터 엔진**: 공매도 과열 + 펀더멘탈 양호 조건 조합 필터
- **숏커버 감지**: 공매도 감소 + 거래량 급증 + 가격 반등 동시 발생 감지
- **통계 엔진**: 과거 숏스퀴즈 사례 수집 → D+3/D+5/D+10 수익률 통계
- **알림**: 숏커버 시작 징후 감지 시 텔레그램 전송
- **GitHub Actions**: 장 마감 후 공매도 데이터 갱신 + 후보 리스트 업데이트

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (공매도 데이터 이미 수집 중, 통계 로직 추가)
- **기대효과**: 공매도에 의한 과매도 구간에서 역발상 매수 기회 포착. 숏스퀴즈 초기 진입으로 단기 수익 극대화. 기존 매수 신호에 "공매도 해소" 차원을 추가하여 확신 강화.

## 아이디어 17: 실적 서프라이즈 프리뷰

### 개요
실적 발표 예정 종목에 대해 **발표 전 수급/기술적 신호 변화**를 추적하여, 시장이 "서프라이즈를 미리 반영하고 있는지"를 감지하는 시스템. 실적 발표 전 외국인/기관의 선제적 매수 패턴을 분석하여 서프라이즈 가능성을 예측한다.

### 핵심 가치
- **분석 깊이**: 실적 발표 전 "선반영 패턴"을 데이터로 포착
- **실시간성**: 실적 발표 D-7 ~ D-1 기간의 수급 변화를 매일 업데이트
- **편의성**: "다음 주 실적 발표 종목 중 주목할 것" 자동 브리핑

### 작동 방식 (핵심 분석)

#### 실적 전 선반영 패턴 분석
```
실적 발표 D-7 ~ D-1 기간 관찰 지표:

1. 외국인 순매수 전환 여부
   → 실적 호전 기대 시 외국인이 먼저 매수하는 경향
2. 기관 순매수 규모 변화
   → 증권사 리서치팀은 실적 추정치를 먼저 접근 가능
3. 거래량 변화율 (평소 대비)
   → 조용한 종목에 갑자기 거래량이 늘면 정보 유입 가능성
4. 콜옵션 비율 변화 (파생 가능 종목)
   → 옵션 시장의 포지션 변화가 주식보다 빠름
5. 가격 움직임 (D-10 대비 D-1 수익률)
   → 양의 수익률 + 순매수 = 서프라이즈 선반영 가능성
```

#### 서프라이즈 확률 스코어
```
📊 삼성전자 (005930) — 실적 발표 D-3
━━━━━━━━━━━━━━━━━

서프라이즈 스코어: 78/100 (높음)

┌─────────────────┬──────────┬────────┐
│ 지표             │ 현재     │ 점수   │
├─────────────────┼──────────┼────────┤
│ 외국인 D-7 누적  │ +2,800억  │ 25/25  │
│ 기관 D-7 누적    │ +420억   │ 15/25  │
│ 거래량 변화율    │ +180%    │ 20/25  │
│ D-10 대비 수익률 │ +5.2%   │ 18/25  │
└─────────────────┴──────────┴────────┘

과거 유사 패턴 (스코어 70+):
• 12건 중 9건 실적 서프라이즈 (75%)
• 서프라이즈 시 D+1 평균: +4.2%
• 미스 시 D+1 평균: -3.8%

💡 해석: 외국인 대규모 선제적 매수 + 거래량 급증.
   시장이 실적 호조를 미리 반영하는 패턴.
   다만 이미 +5.2% 상승했으므로 발표 후 차익실현 주의.
```

### 텔레그램 알림 예시
```
📅 [실적 프리뷰] 다음 주 실적 발표 주목 종목
━━━━━━━━━━━━━━━━━

🔴 서프라이즈 가능성 높음 (스코어 70+)
1. 삼성전자 (3/20 발표) — 스코어 78
   외국인 +2,800억 (D-7) | 거래량 +180%
2. SK하이닉스 (3/21 발표) — 스코어 72
   기관 +650억 (D-7) | 가격 +3.8%

🟡 관찰 필요 (스코어 50~69)
3. LG에너지솔루션 (3/19) — 스코어 55
   수급 혼조 | 거래량 소폭 증가

🟢 선반영 약함 (스코어 50 미만)
4. 카카오 (3/22) — 스코어 32
   외국인 순매도 | 거래량 평이

💡 전략: 스코어 70+ 종목은 실적 발표 전
   이미 선반영이 진행 중. 발표 후 추가 상승보다
   차익실현에 주의. 스코어 50 미만 중 펀더멘탈
   양호 종목은 "미반영 서프라이즈" 기회 가능.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 투자자별 매매동향 | theme_analysis → 외국인/기관 순매수 시계열 |
| 종목 시세 히스토리 (90일) | theme_analysis → 실적 발표 전후 수익률 계산 |
| 거래량 프로파일 | theme_analysis → 정상 거래량 기준선 대비 변화율 |

### 기술 구현
- **실적 캘린더**: KRX/DART API 또는 뉴스 크롤링으로 실적 발표 일정 수집
- **선반영 스코어**: D-7 수급/거래량/가격 변화를 가중 평균하여 0~100 스코어 산출
- **과거 패턴 DB**: 실적 발표 전 유사 스코어 대비 실제 서프라이즈 여부 통계 구축
- **알림**: D-7, D-3, D-1에 실적 프리뷰 브리핑 자동 전송
- **GitHub Actions**: 주 1회 실적 캘린더 갱신, 일 1회 스코어 계산

### 난이도 & 기대효과
- **난이도**: ★★★★☆ (실적 캘린더 수집 + 선반영 스코어 설계 + 통계 검증)
- **기대효과**: 실적 시즌의 가장 큰 불확실성을 데이터로 관리. "실적 어떨까?" 대신 "시장은 이미 이렇게 반응하고 있다"는 객관적 근거 제공. 실적 발표 전후 매매 전략 수립의 핵심 도구.

## 아이디어 18: 개인 포트폴리오 리밸런싱 어드바이저

### 개요
보유 종목의 비중/수익률/상관관계를 분석하여 **최적 리밸런싱 시점과 방법**을 AI가 제안하는 시스템. 단순 개별 종목 분석을 넘어 "포트폴리오 전체의 건강 상태"를 진단하고, 비중 조절/종목 교체를 구체적으로 권고한다.

### 핵심 가치
- **분석 깊이**: 개별 종목이 아닌 포트폴리오 레벨의 최적화
- **편의성**: "지금 내 포트폴리오 어떤 상태야?"에 대한 종합 답변
- **실시간성**: 시장 변동에 따른 리밸런싱 필요 시점 자동 감지

### 작동 방식 (핵심 분석)

#### 1. 포트폴리오 건강 진단
```
📊 내 포트폴리오 진단 결과
━━━━━━━━━━━━━━━━━

보유 종목 (5종목):
┌──────────────┬──────┬───────┬──────────┐
│ 종목          │ 비중  │ 수익률 │ 매매 신호 │
├──────────────┼──────┼───────┼──────────┤
│ 삼성SDI      │ 35%  │ +12.5%│ 적극매수  │
│ SK하이닉스   │ 25%  │ +8.2% │ 매수     │
│ 에코프로비엠  │ 20%  │ -3.8% │ 중립     │
│ 한화에어로    │ 12%  │ +5.1% │ 매수     │
│ 카카오       │  8%  │ -7.2% │ 매도     │
└──────────────┴──────┴───────┴──────────┘

⚠️ 진단 결과:
1. 섹터 편중: 2차전지 55% (삼성SDI+에코프로) — 위험
2. 상관관계 과다: 삼성SDI-에코프로 상관계수 0.85
3. 매도 신호 보유: 카카오 (-7.2%, 매도 신호)
4. 승자 과대: 삼성SDI 수익 +12.5%인데 비중 35%
   → 원래 매수 시 비중은 25%였으나 상승으로 비대

포트폴리오 건강도: 58/100 (개선 필요)
```

#### 2. 리밸런싱 제안
```
🔄 리밸런싱 제안:

1. 카카오 전량 매도 (매도 신호 + 손실 -7.2%)
   → 회수 자금으로 방산 섹터 편입 검토 (한화에어로 추가매수)

2. 삼성SDI 비중 축소 (35% → 25%)

## 아이디어 19: 시장 심리 온도계 (센티멘트 인덱스)

### 개요

## 아이디어 20: 종목 상관관계 네트워크 맵

### 개요
TOP 100 종목 간의 **가격 상관관계를 네트워크 그래프로 시각화**하는 시스템. 어떤 종목이 같이 움직이고, 어떤 종목이 독립적으로 움직이는지를 한눈에 파악. 예상치 못한 종목 간 연결고리(hidden correlation)를 발견하고, 상관관계 변화를 추적한다.

### 핵심 가치
- **분석 깊이**: 테마 분류를 넘어 실제 가격 움직임 기반의 종목 관계 분석
- **편의성**: 복잡한 상관 행렬 대신 직관적 네트워크 시각화
- **실시간성**: 상관관계 급변(디커플링/커플링) 감지 시 알림

### 작동 방식 (핵심 분석)

#### 상관관계 네트워크 시각화
```

• 색상 = 섹터 (같은 색 = 같은 테마)
• 거리 = 상관관계 역수 (높을수록 가까이)

┌──────────────────────────────────────┐
│                                      │
│    [삼성SDI]──────[에코프로비엠]      │
│       │  ╲           │              │

│       │    ╲        │              │
│    [SK하이닉스]  [한미반도체]         │
│       │          │                  │ ← AI반도체 클러스터

│                                      │
└──────────────────────────────────────┘
```

#### 상관관계 변화 감지
```

1. 디커플링 (상관관계 급락):
   카카오 ↔ 네이버: 0.82 → 0.35 (30일)
   → 원인: 카카오 실적 부진 vs 네이버 AI 모멘텀
   → 의미: 더 이상 "인터넷" 섹터로 묶어서 볼 수 없음

2. 숨은 커플링 (예상 밖 상관관계 발견):
   한화에어로 ↔ 현대로템: 0.23 → 0.78 (30일)
   → 원인: 방산 수출 모멘텀 동시 수혜
   → 의미: 방산 테마 확장 — 한화에어로 투자자는 현대로템도 주시

3. 클러스터 변화:
   SK하이닉스가 "AI반도체" → "삼성전자 그룹"으로 이동
   → 삼성전자와의 상관 0.55 → 0.82 (반도체 업황 동조)
```

## 아이디어 21: 갭 분석 & 갭 매매 전략 엔진

### 개요
매일 장 시작 시 발생하는 **가격 갭(전일 종가 vs 당일 시가 괴리)**을 체계적으로 분석하는 시스템. 갭의 유형(돌파 갭/일반 갭/소진 갭)을 분류하고, 갭 발생 후 "갭이 메워지는지 vs 추세가 이어지는지"를 과거 통계로 예측한다.

### 핵심 가치
- **실시간성**: 장 시작 직후 (09:00~09:05) 갭 발생 종목 즉시 알림
- **분석 깊이**: 갭 유형 분류 + 갭 후 움직임 통계 기반 전략 제시
- **편의성**: "오늘 갭 상승 종목 중 추가 상승 가능성 높은 것" 자동 선별

### 작동 방식 (핵심 분석)

#### 갭 유형 분류
```
갭 분류 기준 (시가 vs 전일 종가):

1. 돌파 갭 (Breakaway Gap)
   조건: 갭 +3% 이상 + 거래량 300%+ + 신고가 돌파
   의미: 강한 추세 시작 → 매우지지 않을 가능성 높음
   과거 통계: 갭 유지율 72% (D+5 기준)

2. 일반 갭 (Common Gap)
   조건: 갭 +1~3% + 거래량 평이
   의미: 일시적 수급 불균형 → 메워질 가능성
   과거 통계: 갭 메워짐 확률 65% (D+3 기준)

3. 소진 갭 (Exhaustion Gap)
   조건: 5일+ 상승 후 갭 상승 + 거래량 정점
   의미: 상승 마무리 → 되돌림 가능성 높음
   과거 통계: D+3 하락 확률 70%

4. 갭 하락 (Gap Down)
   조건: 시가 < 전일 종가
   세부 분류: 패닉 갭(-5% 이상) / 일반 갭 하락
```

- **GitHub Actions**: 장 시작 직후 트리거 (09:05 KST)

- **난이도**: ★★★☆☆ (갭 감지 로직 간단, 유형 분류 + 통계 구축 필요)
- **기대효과**: 장 시작 매매의 핵심 정보를 체계적으로 제공. "갭 상승했는데 사도 돼?"에 통계적 답변. 돌파 갭 추종 + 소진 갭 차익실현 전략으로 수익 극대화.

## 아이디어 22: 배당 & 밸류에이션 스크리너

### 개요
성장주 위주의 기존 분석을 보완하여 **가치투자 관점의 종목 발굴 도구**를 제공하는 시스템. PER/PBR/배당수익률/ROE 등 밸류에이션 지표를 종합하여 "저평가 우량주"를 자동 선별하고, 성장 모멘텀(테마/신호)과 결합하여 "성장 + 가치" 겸비 종목을 찾는다.

### 핵심 가치
- **분석 깊이**: 기술적 분석(신호) + 펀더멘탈(밸류에이션) 통합 관점
- **편의성**: "저평가인데 모멘텀도 있는 종목" 자동 선별
- **실시간성**: 주가 변동에 따른 밸류에이션 변화 일별 업데이트

### 작동 방식 (핵심 분석)

#### 밸류에이션 스코어 산출
```
밸류에이션 스코어 구성 (0~100):

┌──────────────┬──────┬────────────────────┐
│ 지표          │ 비중  │ 평가 기준           │
├──────────────┼──────┼────────────────────┤
│ PER 상대순위  │ 25%  │ 업종 내 하위 → 고점수│
│ PBR 상대순위  │ 20%  │ 업종 내 하위 → 고점수│
│ 배당수익률    │ 20%  │ 높을수록 고점수      │
│ ROE (추정)    │ 20%  │ 높을수록 고점수      │
│ 시총 대비 매출 │ 15%  │ PSR 낮을수록 고점수  │

- **매트릭스**: 밸류 스코어 × 모멘텀 스코어(매매 신호 + 수급) 2차원 매핑
- **가치함정 필터**: 밸류 높음 + 매도 신호 = 가치함정 경고 태그

- **알림**: 주 1회 밸류 리포트 + 스코어 급변 시 수시 알림
- **GitHub Actions**: 일 1회 밸류 스코어 갱신

### 난이도 & 기대효과
- **난이도**: ★★☆☆☆ (PER/PBR 이미 수집 중, 정규화 + 스코어링만 필요)
- **기대효과**: 모멘텀/기술적 분석 편향을 보완하는 펀더멘탈 관점 추가. "싸고 좋은 주식"을 체계적으로 발굴. 가치함정 경고로 저PER 함정 회피.

## 아이디어 23: 거래량-가격 괴리 분석기 (Volume-Price Divergence)

### 개요
주가는 상승하는데 거래량이 줄어들거나, 주가는 하락하는데 거래량이 급증하는 등 **가격과 거래량의 괴리(다이버전스)**를 체계적으로 탐지하는 시스템. 거래량-가격 괴리는 추세 전환의 가장 강력한 선행 지표 중 하나로, 이를 자동화하여 "곧 꺾일 종목"과 "곧 반등할 종목"을 사전에 포착한다.

### 핵심 가치
- **분석 깊이**: 가격 단독이 아닌 가격+거래량 복합 분석으로 추세 전환 예측
- **실시간성**: 일별/분봉 단위 괴리 감지, 강한 괴리 즉시 알림
- **편의성**: "상승 중인데 거래량이 빠지는 위험 종목" 자동 리스트

### 작동 방식 (핵심 분석)

#### 4가지 핵심 괴리 패턴
```
패턴 1: 약세 다이버전스 (매도 준비)
┌──────────────────────────────────┐
│ 가격:  ╱╲     ╱╲  ← 고점 갱신   │
│       ╱  ╲   ╱  ╲              │
│      ╱    ╲ ╱    ╲             │
│ 거래량: ████  ██   ← 감소       │
│                                  │
│ → 가격은 오르는데 거래량 감소      │
│ → "사는 사람이 줄어들고 있다"      │
│ → 고점 부근 매도 검토             │
└──────────────────────────────────┘

패턴 2: 강세 다이버전스 (매수 준비)
┌──────────────────────────────────┐
│ 가격:  ╲    ╲   ← 저점 갱신     │
│         ╲╱   ╲╱                 │
│ 거래량: ██   ████  ← 증가       │
│                                  │
│ → 가격은 내리는데 거래량 증가      │
│ → "바닥에서 누군가 매집 중"        │
│ → 반등 진입 검토                 │
└──────────────────────────────────┘

패턴 3: 거래량 고갈 경고
• 5일 연속 거래량 감소 (50% 이상)
• 가격 횡보 또는 미약한 상승
→ "에너지 소진, 방향성 폭발 임박"

패턴 4: 거래량 선행 급증
• 가격 변동 없이 거래량만 300%+ 급증
• 아직 가격에 반영되지 않은 정보 유입 가능
→ "누군가 먼저 알고 사고 있다"
```

```

## 아이디어 24: AI 투자 멘토 (학습형 피드백 시스템)

### 개요
사용자의 실제 매매 패턴을 장기간 추적하여 **개인화된 투자 습관 분석 및 개선 코칭**을 제공하는 시스템. 매매 일지(#8)를 넘어, "당신은 이런 상황에서 항상 이런 실수를 한다"는 반복 패턴을 AI가 학습하고, 같은 상황이 다시 발생할 때 사전 경고를 보낸다.

### 핵심 가치
- **분석 깊이**: 단발성 회고가 아닌 장기 매매 패턴 학습 + 반복 실수 탐지
- **편의성**: 같은 실수 반복 방지를 위한 실시간 사전 경고
- **실시간성**: "지금 이 상황, 과거에 비슷했을 때 당신은 이렇게 했고 결과는 이랬다"

### 작동 방식 (핵심 분석)

#### 매매 패턴 프로파일 구축
```
📊 내 투자 프로파일 (최근 90일 매매 기반)
━━━━━━━━━━━━━━━━━

매매 성향:
• 평균 보유 기간: 3.2일 (단타 성향)
• 매수 시간대: 오전 10시 집중 (62%)
• 선호 섹터: 2차전지 (38%), AI반도체 (28%)
• 평균 매매 빈도: 주 8.3회

강점 패턴 (승률 70%+):
✅ 돌파 갭 + 테마 대장주 매수 → 승률 82% (+4.8%)
✅ 외국인 3일 순매수 + 적극매수 → 승률 78% (+3.5%)
✅ 오전 횡보 후 2시 이후 진입 → 승률 71% (+2.1%)

약점 패턴 (승률 40% 이하):
❌ 급등 후 추격 매수 (+5% 이상에서 진입) → 승률 28% (-3.2%)
❌ 손절 라인 미준수 (설정 -3%인데 -7%까지 보유) → 38%
❌ 금요일 오후 충동 매수 → 승률 35% (-2.8%)
❌ 하락장에서 물타기 → 승률 22% (-8.5%)
```

#### 실시간 사전 경고
```
🧠 [AI 멘토] 과거 패턴 경고
━━━━━━━━━━━━━━━━━

현재 상황: 에코프로 +6.2% 급등 중
당신의 행동 예측: 추격 매수 시도 가능성

⚠️ 과거 유사 상황 (급등 종목 추격 매수):
• 7건 시도 중 5건 손실 (승률 28%)
• 평균 결과: -3.2%
• 최악의 경우: 클래시스 -11.5% (2026-02-18)

📊 당시 vs 현재 비교:
• 당시 에코프로 패턴: 3일 +12% 후 +6% 추가 급등
• 현재 에코프로: 3일 +14% 후 +6.2% 급등 (매우 유사)
• 당시 결과: 매수 다음날 -4.8% 조정

💡 멘토 코칭:
"급등 추격 매수는 당신의 반복적 약점 패턴입니다.
 과거 7건 중 5건이 손실이었습니다.
 추천: 장 마감 후 냉정하게 재평가.
 진입하려면 최소 -2% 눌림목 대기."
━━━━━━━━━━━━━━━━━
```

#### 월간 멘토 리포트
```
📋 [월간 멘토] 2026년 3월 투자 성적표
━━━━━━━━━━━━━━━━━

총 매매: 35건 | 승률: 62% | 평균수익: +1.8%

🏆 이번 달 최고의 매매:
한화에어로 +9.2% (돌파 갭 + 강점 패턴 활용)

💔 이번 달 최악의 매매:

• 손절 준수율: 2월 45% → 3월 68% (개선 중)

🎯 다음 달 목표:

━━━━━━━━━━━━━━━━━
```

## 아이디어 25: 장전/장후 프리마켓 모니터

### 개요
정규장(09:00~15:30) 외의 **시간외 거래(장전 08:30~09:00, 장후 15:40~18:00)**와 **야간 시장 이벤트(미국 증시, 환율, 원자재)**를 모니터링하여, "오늘 장은 어떻게 시작할 것인가"를 장 시작 전에 예측하는 시스템.

- **실시간성**: 장 시작 전 가장 중요한 30분(08:30~09:00)의 정보를 선제적 제공
- **분석 깊이**: 시간외 거래 + 해외 시장 + 환율 + ADR을 종합한 개장 예측
- **편의성**: 아침에 한 메시지만 읽으면 오늘 장 분위기 파악 완료

### 작동 방식 (핵심 분석)

```

   • S&P 500, NASDAQ, SOX(반도체) 등락률
