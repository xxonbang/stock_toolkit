## 1. ATR(Average True Range) 기반 손절 전략

### 1.1 ATR 계산 공식

```
True Range = max(
  |당일 고가 - 당일 저가|,
  |당일 고가 - 전일 종가|,
  |당일 저가 - 전일 종가|
)

ATR(n) = n일간 True Range의 이동평균 (일반적으로 SMA 또는 EMA)
```

### 1.2 ATR 기반 손절가 설정

```
매수 포지션 손절가 = 진입가 - (ATR × 배수)
매도 포지션 손절가 = 진입가 + (ATR × 배수)
```

### 1.3 배수(Multiplier)별 특성

| 배수 | 용도 | 장점 | 단점 |
|------|------|------|------|
| 1.0x | 스캘핑/초단타 | 빠른 손절, 손실 최소화 | 휩소(whipsaw) 빈발, 수익 -60bp 감소 |
| 1.5x | 단타/데이트레이딩 | 조용한 시장에서 적절 | 변동성 큰 시장에서 조기 손절 |
| 2.0x | 범용 (가장 보편적) | 균형잡힌 손익비, 최대낙폭 32% 감소(1000거래 연구) | 중간 수준의 손실 허용 필요 |
| 2.5~3.0x | 스윙/추세 추종 | 추세 유지력 우수, 수익 15% 향상(고정 대비) | 넓은 손절폭 = 큰 단일 손실 가능 |
| 3.0x+ | 장기 추세/포지션 | 노이즈 필터링 극대화 | 손실 발생 시 금액 큼 |

### 1.4 ATR 기간(Period) 설정 가이드

| 트레이딩 스타일 | 차트 타임프레임 | 권장 ATR 기간 | 이유 |
|----------------|----------------|--------------|------|
| 스캘핑 (5분봉) | 5분 | 5~10 | 빠른 반응, 최근 변동성 반영 |
| 데이트레이딩 | 15분~1시간 | 10~14 | 반응성과 안정성 균형 |
| 스윙 트레이딩 | 일봉 | 14~21 | 안정적 변동성 측정 |
| 포지션 트레이딩 | 주봉 | 20~21 | 장기 노이즈 필터링 |

**핵심**: ATR(14)는 범용 기본값이나, 일중 단타에서는 ATR(10)이 더 반응성 높고 적절함. ATR(5)는 노이즈가 많아 단독 사용 비권장.

## 2. Chandelier Exit (샹들리에 엑시트)

### 2.1 개요

Charles Le Beau가 개발, Alexander Elder의 저서에서 소개. 추세 추종 트레일링 스톱.

### 2.2 공식

```
롱 포지션: Chandelier Exit = n기간 최고가 - (ATR(n) × 배수)
숏 포지션: Chandelier Exit = n기간 최저가 + (ATR(n) × 배수)
```

### 2.3 기본 파라미터

- **기간(n)**: 22일 (일봉 기준, 약 1개월 영업일)
- **배수**: 3.0x ATR
- **작동 원리**: 최고가 갱신 시 손절선도 상향 → 추세 중 이익 보호 + 조기 이탈 방지

### 2.4 파라미터 조정 가이드

| 시장 상황 | 기간 | 배수 | 이유 |
|-----------|------|------|------|
| 고변동성 주식 | 14~22 | 3.0~4.0x | 넓은 버퍼로 휩소 방지 |
| 안정적 추세 | 22 | 2.0~3.0x | 표준 설정 |
| 저변동성 주식 | 22 | 1.5~2.0x | 감도 향상 |

## 3. 고변동성 모멘텀/테마주 전용 전략

### 3.1 갭상승(10~20%+) 종목 대응

갭상승 종목은 일반 ATR 기반 손절이 부적합할 수 있음. 갭 이전의 ATR은 현재 변동성을 과소 반영.

**권장 접근법:**
1. **갭 발생 후 ATR 재계산**: 갭 당일 분봉 데이터로 인트라데이 ATR 별도 산출
2. **갭 기준 손절**: 갭 시작가(당일 시가) 하회 시 즉시 손절
3. **분봉 ATR 기반**: 5분봉 ATR(10) × 2.0~2.5x를 당일 최고가 기준 트레일링
4. **거래량 확인**: 갭상승 후 거래량 급감 시 추가 손절 조건으로 활용

### 3.2 코스닥 테마주 특수 고려사항

- **진입가 대비 -5%~-10%** 구간을 1차 손절 기준으로 설정 (실전 가이드 기준)
- 기계적 실행 필수 — 감정 개입 시 손실 확대
- 테마 소멸 징후(뉴스 소멸, 거래량 급감) 시 ATR과 무관하게 즉시 청산 고려

## 4. 학술 연구 결과: 최적 손절 비율

### 4.1 "Taming Momentum Crashes" (Han, Zhou, Zhu, 2016)

미국 시장 1926~2013년 데이터 기반 대규모 연구.

| 손절 수준 | 샤프 비율 | 최대 월간 손실 | 비고 |
|-----------|----------|---------------|------|
| 손절 없음 (기존 모멘텀) | 0.17 | -49.8% | 기준선 |
| 10% 손절 | 0.50 | -11.4% (동일가중) / -15.4% | 최고 샤프 |
| 15% 손절 | 0.40 | -17.4% | 최적 균형점 |
| 20% 손절 | 0.32 | -16.8% | - |

**핵심 발견:**
- 10% 손절: 샤프 비율 **3배 향상**, 최대 손실 **-49.8% → -11.4%**로 대폭 감소
- 15% 손절: 월 수익률 1.93% (기존 0.99%의 **약 2배**), 위험-보상 최적 균형
- 모멘텀 전략에서 손절은 단순히 손실을 줄이는 것이 아니라 **수익도 향상**시킴

### 4.2 변동성 레짐별 손절 효과

- 변동성이 높을수록 손절 전략의 효과가 더 크게 나타남
- 최저 변동성 포트폴리오: 기존 모멘텀 월 0.46%, 손절 적용 시 0.61%
- 5%~10% 손절 수준에서 **96%의 종목**이 매수-보유 대비 최대 **30배 낮은 분산** 기록

### 4.3 MIT 연구 — "When Do Stop-Loss Rules Stop Losses?"

- 손절 전략은 자산 가격이 장기 의존성(long-range dependence)을 보일 때 매수-보유 대비 양의 수익 프리미엄 발생
- 최적 손절 임계값은 자산의 허스트 지수(Hurst parameter), 평균 수익률, 변동성에 의해 결정

## 5. 손절 방식 비교 연구

### 5.1 방식별 성능 비교

| 방식 | 장점 | 단점 | 적합 전략 |
|------|------|------|----------|
| **고정 %** (예: -2%) | 단순, 구현 쉬움 | 변동성 무시, 조용한 장에서 너무 넓고 변동성 장에서 너무 좁음 | 단순 시스템 |
| **ATR 기반** | 변동성 자동 반영 | 급변 시 지연, 배수 최적화 필요 | 범용, 추세 추종 |
| **ATR 트레일링** | 이익 보호 + 변동성 적응 | 횡보장에서 조기 이탈 | 추세 추종 |
| **시간 기반** | 뉴스/이벤트 리스크 관리 | 가격 무관하게 청산 | 이벤트 드리븐 |
| **하이브리드** | 다중 조건으로 견고 | 복잡, 과최적화 위험 | 정교한 알고리즘 |

### 5.2 87개 손절 전략 백테스트 결과 (Paper to Profit 연구)

SPY, AAPL, QQQ 등 8개 자산 대상:

| 전략 | 상승 수익 포착 (bp) | 하락 회피 (bp) |
|------|---------------------|---------------|
| Parabolic SAR (기준선) | ~0 (음수) | ~30 |
| ATR 트레일링 (1x) | -60 (손실) | - |
| 고정 % 손절 | ~0 | ~6 |
| 피봇 포인트 손절 | - | 83 |
| 이동평균 교차 손절 | 130 | 79 |
| 켈트너 채널 손절 | 211 | 59 |
| 과거 분위수 손절 | 375 | 113 |

**결론**: 단순 ATR 1x는 오히려 해로움. 켈트너 채널, 과거 분위수 기반이 최우수 성과.

### 5.3 Build Alpha 연구 (2006~2022)

- RSI 전략에 고정 1% 손절 추가 시 순이익이 **절반으로 감소**
- 반면 추세 추종 전략에서는 손절이 성과 개선
- **핵심**: "최고의 손절은 없다. 전략과 자산에 따라 완전히 다르다"
- 평균 회귀 전략에서는 손절이 성과를 **악화**시키는 경우가 많음
- 추세 추종 전략에서는 손절이 성과를 **개선**하는 경우가 많음

## 6. 휩소(Whipsaw) 문제와 대응

### 6.1 휩소란?

가격이 한 방향으로 이동하여 손절을 트리거한 후 즉시 반대 방향으로 돌아가는 현상. 타이트한 손절일수록 빈발.

### 6.2 휩소 취약 조건 식별

- 거래량 감소 + 좁은 레인지 → 횡보 구간
- 다중 타임프레임 시그널 충돌
- ADX < 20 (추세 부재)
- 변동성 급증 직전 (볼린저 밴드 수축)

### 6.3 휩소 방지 전략

1. **ATR 적응형 스톱**: 14기간 ATR × 2.0x 이상 → 정상 변동 범위 허용
2. **ADX 필터**: ADX > 25일 때만 모멘텀 매매 진입 → 휩소 노출 대폭 감소
3. **다중 타임프레임 확인**: 5분봉 + 15분봉 + 60분봉 방향 일치 시만 진입
4. **시간 지연 트레일링**: 유리한 가격이 일정 시간 유지되어야 손절선 이동
5. **단계적 트레일링 (Stepped)**: 연속 이동 대신 일정 간격으로 단계적 이동 → 거래 비용 절감

## 8. 실전 구현 권장안 (우리 시스템 적용)

### 8.1 하이브리드 동적 손절 모델

```python
def calculate_dynamic_stoploss(entry_price, atr_value, stock_type, adx_value):
    """
    하이브리드 동적 손절가 계산

    stock_type: 'large_cap' | 'mid_cap' | 'small_theme' | 'gap_up'
    """
    # 종목 유형별 ATR 배수
    multipliers = {
        'large_cap': 2.0,
        'mid_cap': 2.5,
        'small_theme': 3.0,
        'gap_up': 2.0,  # 분봉 ATR 사용 전제
    }

    # 최대 손절폭 제한 (절대값)
    max_loss_pct = {
        'large_cap': 0.05,
        'mid_cap': 0.07,
        'small_theme': 0.10,
        'gap_up': 0.10,
    }

    multiplier = multipliers[stock_type]

    # ADX 기반 배수 조정: 추세 약하면 타이트하게, 강하면 넓게
    if adx_value < 20:
        multiplier *= 0.8  # 횡보장 → 타이트
    elif adx_value > 40:
        multiplier *= 1.2  # 강한 추세 → 여유

    # ATR 기반 손절가
    atr_stop = entry_price - (atr_value * multiplier)

    # 최대 손절폭 제한 적용
    max_stop = entry_price * (1 - max_loss_pct[stock_type])

    # 둘 중 높은 값 (= 더 타이트한 손절) 적용
    stop_price = max(atr_stop, max_stop)

    return stop_price
```

### 8.2 트레일링 스톱 로직

```python
def update_trailing_stop(current_stop, highest_price, atr_value, multiplier):
    """
    Chandelier Exit 방식 트레일링 스톱
    손절선은 오직 상향만 가능 (하향 불가)
    """
    new_stop = highest_price - (atr_value * multiplier)
    return max(current_stop, new_stop)  # 기존보다 높을 때만 갱신
```

### 8.3 전략 유형별 손절 적용 원칙

| 전략 | 손절 방식 | 이유 |
|------|----------|------|
| 모멘텀/추세 추종 | ATR 트레일링 (2.5~3.0x) | 추세 유지 + 이익 보호 |
| 평균 회귀 | 고정 % (-3~5%) 또는 손절 미적용 | 손절이 오히려 성과 악화 |
| 갭상승 데이트레이딩 | 분봉 ATR + 시가 기준 하드 스톱 | 갭 실패 시 빠른 탈출 |
| 테마주 단기 | ATR(10) × 2.5x + 최대 -10% 캡 | 고변동성 허용 + 대형 손실 방지 |

**Type A — 지속형 급등 (에코플라스틱)**

**Type B — 소멸형 급등 (미래에셋벤처투자)**

### 10.3 발견된 연구 보완점 (10건)

#### 보완점 1: "넓은 손절 = 항상 유리"는 오류

기존 연구 결론 #4 "하이브리드: ATR 동적 + 절대값 캡이 가장 실용적"은 **V자 회복 패턴에서만 유효**하다.

미래에셋벤처투자처럼 **고점 후 지속 하락하는 패턴**에서는:
- -2% 손절: 손실 -2.0%로 제한 (최선)
- -10% 손절: 종가까지 -6.3% (더 큰 손실)
- 홀딩: -6.3% (최악)

**수정안**: 손절폭을 결정할 때 **패턴 분류(V자 회복 vs 우하향)**가 선행되어야 함. 단순히 변동성만으로 손절폭을 결정하면 안 됨.

#### 보완점 2: 거래량 추이를 손절 판단에 반영 필요

| 시간 | 에코플라스틱 거래량 | 미래에셋벤처투자 거래량 |
|------|-------------------|---------------------|
| 09:30 | 28.8M | 7.0M |
| 12:00 | 1.6M | 981K |
| 15:00 | 3.7M (**반등**) | 815K |
| 15:30 | 3.7M (유지) | 593K (**급감**) |

에코플라스틱: 오후 거래량이 다시 증가 → 매수세 유입 → 회복 가능성 높음
미래에셋벤처투자: 거래량이 지속 감소 → 매수세 소멸 → 회복 가능성 낮음

기존 연구 섹션 3.1 #4에서 "거래량 급감 시 추가 손절 조건"을 언급했으나, **구체적 구현 로직이 없었음**.

**수정안**: 거래량 기반 손절 트리거 추가
```
if 최근 30분 거래량 < 09:30 거래량의 10% AND 현재가 < 진입가:
    즉시 손절 (거래량 소멸 = 회복 동력 부재)
```

#### 보완점 3: Han et al. 연구의 적용 범위 오해

기존 문서에서 Han, Zhou, Zhu(2016)의 "10% 손절 최적"을 인용하며 당일 매매에 적용 가능한 것처럼 기술했으나, 이 연구는:
- **월간 리밸런싱** 기반 (일중 매매 아님)
- **포트폴리오 수준** 손절 (개별 종목 아님)
- **수개월 보유** 기간 전제

**수정안**: 학술 연구 결과의 적용 범위를 명시하고, 일중 매매에는 별도 기준이 필요함을 강조.

#### 보완점 4: 시간대별 손절 전략 차등 필요

기존 문서 섹션 7.4에서 시간대별 변동성을 기술했으나, **손절 전략에 시간을 반영하는 구체적 방안이 없음**.

실제 패턴: 두 종목 모두 **11:00에 고점**, 이후 하락.
- 오전(09:30-11:00): 모멘텀 진입 구간 → 넓은 손절 (회복 시간 충분)
- 오후(13:00 이후): 추세 확정 구간 → 타이트한 손절 (장 마감까지 회복 시간 부족)

**수정안**: 시간대별 손절 배수 조정
```
09:30~11:00: ATR × 3.0x (모멘텀 형성 구간, 넓은 여유)
11:00~13:00: ATR × 2.5x (추세 확인 구간)
13:00~14:30: ATR × 2.0x (오후 재진입, 시간 제한적)
14:30~15:30: ATR × 1.5x 또는 손절 타이트 (장 마감 임박)
```

#### 보완점 5: 하이브리드 모델에 누락된 입력 변수

기존 `calculate_dynamic_stoploss(entry_price, atr_value, stock_type, adx_value)` 모델의 입력이 4개뿐임. 실전에서 중요한 변수 누락:

| 누락 변수 | 중요성 | 에코플라스틱 | 미래에셋벤처투자 |
|-----------|--------|------------|----------------|
| **거래량 추이** | 높음 | 반등 (양호) | 급감 (위험) |
| **진입 시각** | 높음 | 오전 = 회복 여유 | 오전 = 회복 여유 |
| **시가 대비 현재가** | 중간 | 시가 위 유지 | 시가 하회 (15:00) |
| **전일 대비 등락률** | 중간 | +24% (극단적) | +10% (보통) |

#### 보완점 6: "모든 급등주에 동일 전략" 가정의 한계

기존 문서의 종목 분류가 `large_cap / mid_cap / small_theme / gap_up` 4가지이나, 같은 `gap_up` 내에서도:
- **Type A 지속형** (에코플라스틱): 거래량 유지, V자 회복 → 넓은 손절 유리
- **Type B 소멸형** (미래에셋벤처투자): 거래량 급감, 우하향 → 타이트한 손절 유리
- **Type C 횡보형** (HD현대에너지솔루션): 거래량 극단 급감, 좁은 횡보 → **매수 자체 재고 필요**

이를 구분하지 않으면 손절 전략이 1/3만 맞게 됨.

#### 보완점 7: 갭 크기 대비 리스크/리워드 필터 부재

기존 연구에서 "갭상승 종목" 진입을 전제로 손절만 논의했으나, **진입 여부 자체를 걸러야 하는 경우**가 누락됨.

| 종목 | 갭 크기 | 장중 변동폭 | 종가 수익률 | 변동폭 대비 수익률 |
|------|---------|-----------|-----------|----------------|
| 에코플라스틱 | +24.7% | ±8.9% | +21.0% | **2.4x** (양호) |
| 미래에셋벤처투자 | +8.9% | ±6.3% | +4.6% | **0.7x** (불리) |
| HD현대에너지솔루션 | +2.7% | ±4.1% | +1.4% | **0.3x** (매우 불리) |

**"변동폭 대비 수익률"이 1.0x 미만이면 리스크 > 리워드**. HD현대에너지솔루션(0.3x)은 갭이 작아서 장중 변동성을 감당할 수익 여지가 없음.

**수정안**: 매수 진입 필터로 **최소 갭 크기 기준** 추가:
- 갭 +5% 미만: 변동성 대비 수익 기대값 낮음 → 진입 제외 또는 매우 타이트한 손절
- 갭 +5~15%: 표준 동적 손절 적용
- 갭 +15%+: 넓은 동적 손절 + 거래량 모니터링

#### 보완점 8: 손절 후 동일종목 재매수(whipsaw) 방지 정책

현재 시스템은 손절 후 같은 종목에 다시 매수 시그널이 발생하면 재매수함. HD현대에너지솔루션 사례에서 이것이 **누적 손실을 극대화**하는 구조적 문제임을 확인.

| 재매수 정책 | HD현대에너지 결과 |
|------------|----------------|
| 무제한 재매수 | 3~5회 손절 × -2% = **-6~10%** |
| 당일 1회만 매수 | 1회 -2%, 이후 진입 차단 = **-2%** |
| 손절 후 재매수 금지 | 1회 -2%, 이후 진입 차단 = **-2%** |
| 홀딩 (손절 없음) | 종가 -2.9% = **-2.9%** |

**수정안**: "당일 손절한 종목은 당일 재매수 금지" 정책이 이미 시스템에 존재(`sold_today_codes`)하지만, **손절과 매도를 구분하지 않을 가능성** 검토 필요. 또한 동일 종목 연속 손절 횟수를 추적하여 2회 이상 손절 시 해당 종목 블랙리스트 등록 고려.

#### 보완점 9: "거래량 급감 = 하락" 가정은 오류 (알테오젠 반례)

v3 모델에서 `volume_ratio < 0.1` 시 즉시 손절(-2%)하는 로직이 있었으나, **알테오젠이 이를 반증함:**

| 지표 | 알테오젠 | HD현대에너지솔루션 |
|------|---------|-----------------|
| 거래량 급감 | 89% (284K→30K) | 97% (669K→17K) |
| 종가 | **+6.28% (양호)** | +1.4% (불량) |
| 해석 | 매도세 소진 → 안정화 | 매수+매도 모두 소멸 → 하락 |

**거래량 급감의 두 가지 의미:**
1. **거래량 급감 + 가격 하락**: 매수세 소멸 → 하락 지속 (HD현대에너지, 미래에셋벤처)
2. **거래량 급감 + 가격 유지**: 매도세 소진 → 안정/횡보 (알테오젠)

**수정안**: 거래량 트리거를 단독으로 사용하지 말고, **거래량 + 가격 추세의 조합**으로 판단:
```
# v3 (잘못됨)
if volume_ratio < 0.1:
    base_pct = -0.02  # 즉시 손절

# v4 (수정)
if volume_ratio < 0.1 AND current_price < entry_price:
    base_pct = -0.02  # 매수세 소멸 + 이미 손실 → 즉시 탈출
# volume_ratio < 0.1 AND current_price >= entry_price:
    # 매도세 소진 + 수익 유지 → 손절하지 않음 (알테오젠 패턴)
```

#### 보완점 10: 시장 전반 하락 시 매수 기준 상향 필터 부재

당일 삼성전자(-4.7%), SK하이닉스(-6.2%)가 동반 하락한 **시장 약세일**. 이 환경에서:
- 갭 +24.7% (에코플라스틱) → 종가 +21.0% (방어)
- 갭 +8.9% (미래에셋벤처) → 종가 +4.6% (약화)
- 갭 +2.7% (HD현대에너지) → 종가 +1.4% (거의 소멸)

**시장 하락일에 갭 크기가 작을수록 하락 시장에 흡수됨.**

기존 연구에 시장 맥락 필터가 완전히 부재. 매수 진입 결정 시 시장 지수(KOSPI/KOSDAQ) 상태를 반영해야 함.

**수정안**: 매수 진입 시 시장 필터 추가:
```
# 시장 약세 감지
if KOSPI_change < -1.5% OR KOSDAQ_change < -2.0%:
    min_gap_threshold = 10%  # 기본 5% → 10%로 상향
    max_stop_loss = -3%      # 기본 -5% → -3%로 타이트하게
```

이 필터가 있었다면 HD현대에너지솔루션(+2.7%)과 미래에셋벤처투자(+8.9%)는 시장 약세일에 진입 기준 미달로 **매수 자체가 차단**되었을 것.

### 10.4 수정된 하이브리드 모델 제안 (v2 → v3 → v4)

v2에서 발견된 추가 문제: **갭 크기가 작은 종목의 진입 자체를 필터링하지 않음**.

```python
def should_enter_trade(change_from_prev_close_pct: float, volume_09_30: int) -> bool:
    """
    매수 진입 여부 사전 필터
    - 갭 +5% 미만: 변동성 대비 수익 기대값 낮음 → 진입 불가
    - 갭 +5% 이상이라도 09:30 거래량이 너무 낮으면 유동성 부족 → 진입 불가
    """
    if change_from_prev_close_pct < 5.0:
        return False  # HD현대에너지솔루션 같은 저갭 종목 필터링
    if volume_09_30 < 500_000:  # 50만주 미만: 유동성 부족
        return False
    return True


def calculate_dynamic_stoploss_v3(
    entry_price: int,
    change_from_prev_close_pct: float,  # 전일 대비 등락률
    entry_hour: int,                     # 진입 시각 (시)
    volume_ratio: float,                 # 현재 30분 거래량 / 09:30 거래량
) -> float:
    """
    3종목 실전 검증 기반 동적 손절가 (v3)

    개선점 (v2 대비):
    - 갭 크기별 3단계 손절폭 (v2와 동일)
    - 시간대 조정 (v2와 동일)
    - 거래량 급감 시 즉시 손절 트리거 (v2에서 손절폭 축소 → v3에서 즉시 손절)
    - 절대 하한/상한 캡 유지
    """
    # 1) 등락률 기반 기본 손절폭
    if change_from_prev_close_pct >= 15:
        base_pct = -0.08   # Type A 급등 (에코플라스틱급): -8%
    elif change_from_prev_close_pct >= 5:
        base_pct = -0.05   # Type B 중등 (미래에셋벤처급): -5%
    else:
        base_pct = -0.03   # Type C 소등 (HD현대에너지급): -3%

    # 2) 시간대 조정: 오후로 갈수록 타이트 (회복 시간 부족)
    if entry_hour >= 14:
        base_pct *= 0.6    # 14시 이후: 40% 축소
    elif entry_hour >= 13:
        base_pct *= 0.8    # 13시 이후: 20% 축소

    # 3) 거래량 급감 → 즉시 손절 트리거 (v3 핵심 변경)
    #    v2: 손절폭만 축소 → 여전히 넓은 손절 가능
    #    v3: 거래량 10% 미만이면 최소 손절폭(-2%) 강제 적용
    if volume_ratio < 0.1:
        base_pct = -0.02   # 매수세 소멸 → 즉시 탈출

    # 4) 절대 하한/상한
    final_pct = max(base_pct, -0.10)  # 최대 -10%
    final_pct = min(final_pct, -0.02)  # 최소 -2%

    return entry_price * (1 + final_pct)
```

### 10.4.1 v3 모델의 3종목 시뮬레이션

| 종목 | 09:30 매수 | 갭 | v3 손절폭 | 거래량 트리거 | 최종 결과 |
|------|-----------|-----|----------|-------------|----------|
| 에코플라스틱 | 4,360 | +24.7% | **-8%** (3,491) | 미발동 (거래량 유지) | 미도달, 종가 -3.0% |
| 미래에셋벤처투자 | 29,800 | +8.9% | **-5%** (28,310) | 13:30 거래량 5% → **-2%로 축소** (29,204) | 10:30경 손절 -2.5% |
| HD현대에너지솔루션 | — | +2.7% | **진입 거부** (갭 <5%) | — | **손실 0%** |

**v3 성과 vs 현재 시스템(-2% 고정):**

| 종목 | 현재 시스템 (추정) | v3 모델 | 개선 |
|------|-------------------|---------|------|
| 에코플라스틱 | whipsaw -4~6% | -3.0% (홀딩) | +1~3% |

| HD현대에너지솔루션 | whipsaw -6~10% | **0% (진입 거부)** | **+6~10%** |

### 10.4.2 v4 모델 (6종목 검증 반영)

v3에서 알테오젠 반례로 발견된 거래량 트리거 오류 수정 + 시장 필터 추가:

```python
def should_enter_trade_v4(
    change_from_prev_close_pct: float,
    kospi_change_pct: float,
    volume_09_30: int,
) -> bool:
    """
    매수 진입 여부 사전 필터 (v4: 시장 맥락 반영)
    """
    # 시장 약세 시 진입 기준 상향
    min_gap = 5.0
    if kospi_change_pct < -1.5:
        min_gap = 10.0  # 시장 하락일: 갭 10% 이상만

    if change_from_prev_close_pct < min_gap:
        return False
    if volume_09_30 < 500_000:
        return False
    return True


def calculate_dynamic_stoploss_v4(
    entry_price: int,
    change_from_prev_close_pct: float,
    entry_hour: int,
    volume_ratio: float,          # 현재 30분 거래량 / 09:30 거래량
    current_price: int,           # v4 추가: 현재가 (거래량+가격 조합 판단)
    kospi_change_pct: float,      # v4 추가: 시장 지수 등락률
) -> float:
    """
    6종목 실전 검증 기반 동적 손절가 (v4)

    v3 대비 변경:
    - 거래량 트리거: 거래량 급감 단독 → 거래량 급감 + 가격 하락 조합
    - 시장 필터: KOSPI 하락 시 손절폭 축소 (타이트하게)
    """
    # 1) 등락률 기반 기본 손절폭
    if change_from_prev_close_pct >= 15:
        base_pct = -0.08   # Type A (에코플라스틱급)
    elif change_from_prev_close_pct >= 5:
        base_pct = -0.05   # Type B/D (미래에셋/알테오젠급)
    else:
        base_pct = -0.03   # Type C (HD현대에너지급)

    # 2) 시간대 조정
    if entry_hour >= 14:
        base_pct *= 0.6
    elif entry_hour >= 13:
        base_pct *= 0.8

    # 3) 시장 약세 조정 (v4 신규)
    if kospi_change_pct < -1.5:
        base_pct *= 0.7    # 시장 하락일: 30% 타이트하게

    # 4) 거래량 + 가격 조합 트리거 (v4 수정: v3의 거래량 단독 판단 오류 수정)
    if volume_ratio < 0.1 and current_price < entry_price:
        # 매수세 소멸 + 이미 손실 → 즉시 탈출 (미래에셋벤처, HD현대에너지 패턴)
        base_pct = -0.02
    # volume_ratio < 0.1 and current_price >= entry_price:
    #   매도세 소진 + 수익 유지 → 손절 안 함 (알테오젠 패턴)

    # 5) 절대 하한/상한
    final_pct = max(base_pct, -0.10)
    final_pct = min(final_pct, -0.02)

    return entry_price * (1 + final_pct)
```

### 10.4.3 v4 모델의 6종목 시뮬레이션

| 종목 | 갭 | v4 진입 필터 | v4 손절폭 | 거래량+가격 트리거 | 최종 결과 |
|------|-----|-----------|----------|-----------------|----------|
| 에코플라스틱 | +24.7% | 통과 | -8% | 미발동 (거래량 유지) | 홀딩, 종가 -3.0% |
| 미래에셋벤처투자 | +8.9% | 시장약세→**거부** (기준 10%) | — | — | **손실 0%** |
| HD현대에너지솔루션 | +2.7% | **거부** (갭 <5%) | — | — | **손실 0%** |
| 알테오젠 | +9.6% | 시장약세→**거부** (기준 10%) | — | — | **손실 0%** |
| 삼성전자 | -1.7% | **거부** (갭 음수) | — | — | — |
| SK하이닉스 | -2.5% | **거부** (갭 음수) | — | — | — |

**v4 성과 비교 (시장 약세일 기준):**

| 모델 | 에코플라 | 미래에셋 | HD현대 | 알테오젠 | 합산 |
|------|---------|---------|-------|---------|------|
| 현재(-2%) | -4~6% | -4~6% | -6~10% | -2~4% | **-16~26%** |
| v3 | -3.0% | -2.5% | 0% | 0% (진입거부) | **-5.5%** |
| **v4** | **-3.0%** | **0%** (시장필터) | **0%** | **0%** (시장필터) | **-3.0%** |

**v4는 시장 약세일에 에코플라스틱(갭 +24.7%)만 진입 허용 → 손실 -3.0%로 최소화.**

단, 알테오젠(+9.6%, 종가 +6.3%)은 시장 필터에 의해 진입 거부됨 → **수익 기회 상실**. 이것이 시장 필터의 trade-off.

### 10.5 수정된 최종 결론 (6종목 검증 반영)

기존 결론을 다음으로 전면 수정 (6종목 검증 반영):

> 1. **손절폭 결정보다 "매수하지 않을 종목"을 거르는 게 더 중요하다.** 6종목 중 v4 모델이 진입을 허용한 건 에코플라스틱 1개뿐. 나머지 5종목을 걸러냄으로써 총 손실을 -16~26% → -3.0%로 줄였다.
>
> 2. **시장 맥락이 개별 종목 손절보다 강력한 필터다.** 삼성전자/SK하이닉스가 -5~6% 하락하는 날, 갭 +10% 미만 급등주는 지속성이 약했다 (미래에셋 +4.6%, HD현대 +1.4%). 시장 약세일에 진입 기준을 갭 +10% 이상으로 상향하면 저품질 매수를 원천 차단.
>
> 3. **"거래량 급감 = 하락"은 절반만 맞다.** 알테오젠은 거래량 89% 급감했지만 종가 +6.28%로 양호했다 (매도세 소진). **거래량 급감 + 가격 하락이 동시에 나타날 때만** 손절 트리거가 유효하다.
>
> 4. **동적 손절은 갭 크기에 비례해야 한다.** 갭 +15% 이상은 -8%, 갭 +5~15%는 -5%, 갭 <5%는 진입 거부.
>
> 5. **시장 필터에는 trade-off가 있다.** 알테오젠(+9.6%, 종가 +6.3%)은 시장 필터에 걸려 진입 거부됨 → 수익 기회 상실. 이것은 시장 필터의 비용이며, "대형 손실 방지 vs 일부 수익 기회 포기"의 균형점이다. 6종목 합산으로는 v4가 압도적으로 유리.
>
> 6. **"손절 후 재매수 금지"가 whipsaw 방지의 핵심이다.** 특히 Type C(횡보) 종목에서 반복 매수-손절은 홀딩보다 2~3배 큰 손실을 만든다.
