## 1. 고정 TP vs Trailing-Only vs Hybrid 비교

### 1.1 고정 TP의 장단점

| 장점 | 단점 |
|------|------|
| 높은 승률 (확실한 이익 실현) | 상승 잠재력 차단 (upside cap) |
| 심리적 안정 (확정 수익) | 강한 추세장에서 기회비용 극대화 |
| 슬리피지 최소화 | 기대값(EV) 감소 가능 |

**핵심 연구 결과:** Optimus Futures 분석에 따르면, 고정 목표가의 가장 큰 단점은 "runner를 잡았을 때 money on the table을 남기는 것"이며, 이것이 고정 목표가 접근의 유일하면서도 가장 큰 약점이다.

### 1.2 Trailing-Only의 장단점

| 장점 | 단점 |
|------|------|
| 상승 추세 전체 포착 가능 | 승률 하락 (round-trip 리스크) |
| 기대 수익 극대화 | 심리적 스트레스 (미실현 이익 반납) |
| 상한가 수익 극대화 | 횡보장에서 조기 청산 |

**Round-trip 리스크 분석:**
- 주가 +15% 상승 → 고정 TP 없음 → 되돌림으로 +0% 또는 -2% 손절
- 이 시나리오의 빈도가 핵심 변수
- 현재 시스템의 -3% trailing은 +15% 도달 후 +12%에서 매도 가능 (round-trip 방지)
- **진짜 위험:** +6% 도달 → TP 미발동 → -2% 손절 = 이전에는 +7% 수익이었을 거래가 -2% 손실로 전환

### 1.3 승률 vs 평균 수익 트레이드오프

| 전략 | 예상 승률 | 평균 수익 (승) | 평균 손실 (패) | 기대값 |
|------|----------|--------------|--------------|--------|
| 고정 TP +7% | ~60% | +7% | -2% | +3.4% |
| Trailing-only -3% | ~40-45% | +12-15% | -2% | +3.0-5.0% |
| Hybrid (분할매도) | ~55% | +10-12% | -2% | +3.7-5.1% |

> 주의: 위 수치는 모멘텀/테마주 특성을 반영한 추정치. 실제 값은 백테스트로 검증 필요.

### 1.4 학술/실증 연구 요약

- **Elm Wealth 연구 (S&P 500, ~100년):** "Cut losses, let profits run" 전략이 연 1.4% 초과 수익, Sharpe Ratio 25% 향상
- **Baur & Dimpfl:** 대형주에서는 "cut losses" 전략의 유의미한 초과 수익 미확인 → 소형 모멘텀주에서 효과 더 클 수 있음
- **Oleg Polakow (200만건 백테스트):** SL/TS/TP 조합별 대규모 비교, 시장 환경에 따라 최적 조합 변동

**결론:** 고정 TP 제거는 "승률 하락 + 평균 수익 증가" 트레이드오프. 모멘텀/테마주처럼 fat-tail 분포를 가진 자산에서는 trailing-only가 기대값 우위.

## 2. 최적 Trailing Stop 파라미터

### 2.1 일반적 권장 수치

| 출처 | 대상 | 권장 Trailing % | 비고 |
|------|------|----------------|------|
| Quant Investing (54년 연구) | 장기 주식 | 15-20% | 월간 체크 기준 |
| Nicolas Darvas | 모멘텀 주식 | ~19% | 전설적 모멘텀 트레이더 |
| 데이트레이딩 일반 | 주식 일중 | 0.5-1% | 스캘핑 기준 |
| 모멘텀 데이트레이딩 | 고변동 주식 | 8-10% | 방향성 강한 종목 |

### 2.2 현재 시스템(-3%) 평가

현재 -3% trailing은 **데이트레이딩 모멘텀 주식** 기준으로 상당히 타이트한 편:

- **장점:** 수익 확보 빠름, 급반전 방어
- **단점:** 정상적 되돌림(pullback)에도 청산 가능 → whipsaw 리스크
- **한국 테마주 특성:** 급등주는 5-10% 내 변동이 빈번 → -3%는 너무 좁을 수 있음

### 2.3 갭상승/강세 종목의 Trailing 전략

**갭업 종목은 더 넓은 trailing이 유리:**
- 갭 10%+ 종목은 일중 5-8% 변동 정상
- -3% trailing 적용 시 정상 pullback에서 조기 청산
- 권장: 갭 크기에 비례한 동적 trailing (예: 갭의 30-50%)

### 2.4 시간 기반 동적 Trailing (Time-Decay Trailing)

장중 시간대별 trailing 조정:

| 시간대 | Trailing % | 근거 |
|--------|-----------|------|
| 09:00-10:00 | -5% (넓게) | 장초반 변동성 극대, 방향 미확정 |
| 10:00-13:00 | -4% | 추세 안정화 구간 |
| 13:00-14:30 | -3% | 오후 추세 재개, 점진 수렴 |
| 14:30-15:20 | -2% (좁게) | 장마감 접근, 수익 확보 우선 |

### 2.5 Stepped Trailing (단계별 손절 이동)

수익 구간별 stop 위치를 상향 조정:

| 도달 수익 | Stop 위치 | 최소 보장 수익 | 설명 |
|----------|----------|-------------|------|
| +0% ~ +5% | -2% (고정 SL) | -2% | 기본 손절 유지 |
| +5% 도달 | 0% (진입가) | 0% (본전) | **breakeven stop** |
| +10% 도달 | +5% | +5% | 수익 절반 확보 |
| +15% 도달 | +10% | +10% | 2/3 확보 |
| +20% 도달 | +15% | +15% | 3/4 확보 |
| +25%+ | peak - 3% | peak-3% | 기존 trailing 적용 |

이 방식의 핵심: **고정 TP를 제거하더라도 수익 구간 진입 후 원금 손실 불가능** → round-trip 리스크 차단.

## 3. 분할 매도 (Partial Profit-Taking) 전략

### 3.1 기본 구조: 50/50 Split

| 단계 | 수량 | 조건 | 효과 |
|------|------|------|------|
| 1차 매도 | 50% | +7% 도달 | 확정 수익 확보 |
| 2차 매도 | 나머지 50% | trailing -3% 또는 장마감 | 상승 잠재력 유지 |

**기대 수익 비교 (상한가 +30% 시나리오):**

| 전략 | 수익 계산 | 총 수익 |
|------|----------|---------|
| 현재 (100% at +7%) | 100% × 7% | **+7.0%** |
| 50/50 Split | 50% × 7% + 50% × 27% | **+17.0%** |
| Trailing-only | 100% × 27% | **+27.0%** |

**기대 수익 비교 (+8% 후 되돌림 시나리오):**

| 전략 | 수익 계산 | 총 수익 |
|------|----------|---------|
| 현재 (100% at +7%) | 100% × 7% | **+7.0%** |
| 50/50 Split | 50% × 7% + 50% × 5% | **+6.0%** |
| Trailing-only | 100% × 5% | **+5.0%** |

### 3.2 실증 근거

- **Option Strategist:** 분할 매도는 부분 이익을 확정하면서도 나머지로 추가 상승 참여 가능
- **단점:** "가격이 부분 매도 이후 계속 상승하면, 부분 매도가 전체 수익을 감소시킴" → 상한가 직행 종목에서 수익 제한
- **Go Markets:** 실전 트레이더들의 가장 보편적 접근은 50% 부분 매도 + 나머지 trailing

### 3.3 변형: 30/30/40 3단계 매도

| 단계 | 수량 | 조건 |
|------|------|------|
| 1차 | 30% | +5% 도달 |
| 2차 | 30% | +10% 도달 |
| 3차 | 40% | trailing -3% 또는 장마감 |

장점: 더 세밀한 수익 확보, 단점: 구현 복잡도 증가 + 수수료 증가

## 4. 한국 시장 특수 고려사항

### 4.1 상한가(+30%) 역학

| 특성 | 영향 |
|------|------|
| +30%에서 가격 잠김 (lock) | trailing stop 무의미, 매도 불가 |
| 상한가 도달 시 매수 주문 대기열 형성 | 팔 수 있으면 팔아야 하는 상황 (유동성 제한) |
| 다음 날 갭업 가능성 | D+1 보유 시 추가 수익 기회 |
| 상한가 직행 종목은 09:00-09:30에 결정 | 이 시간대에 +7% 익절이 발동하면 나머지 +23% 포기 |

### 4.2 테마주 급등 패턴

한국 테마주 급등의 전형적 패턴:
1. **1차 급등 (09:00-09:30):** 동시호가 매물 소화 → +10-15%
2. **1차 조정 (09:30-10:00):** 차익실현 → -3~5% 되돌림
3. **2차 급등 (10:00-11:00):** 기관/세력 추가 매수 → +20-25%
4. **상한가 도전 (11:00-15:00):** +30% 진입 시도

**현재 시스템의 문제:**
- 1차 급등에서 +7% 도달 → 즉시 익절
- 2차 급등, 상한가 도전 참여 불가
- 수익의 77% (= 1 - 7/30) 포기

### 4.3 한국 시장 특유 리스크

| 리스크 | 설명 | 대응 |
|--------|------|------|
| VI (변동성 완화장치) | 5분간 매매 정지 → trailing 의미 없음 | VI 발동 시 trailing 일시 정지 |
| 상한가 후 급반전 | 다음 날 -10~-20% 가능 | D+0 장마감 청산 정책 유지 |
| 작전주/세력 매도 | 급락 후 반등 없이 하한가 | -2% 고정 손절 유지 |

## 5. "Let Winners Run" vs "Take Profit Quickly" 통계 분석

### 5.1 수익 분포 특성 (Fat-tail)

모멘텀/테마주의 수익 분포는 정규분포가 아닌 **right-skewed fat-tail:**

```
빈도
 ▓▓▓▓▓▓▓▓▓▓  -2% (손절, ~40%)
 ▓▓▓▓▓▓       0~+5% (소폭 이익, ~25%)
 ▓▓▓▓         +5~+10% (중간 이익, ~15%)
 ▓▓           +10~+20% (큰 이익, ~10%)
 ▓            +20~+30% (상한가, ~5-10%)
```

고정 TP +7%는 이 분포의 **오른쪽 꼬리를 절단** → fat-tail 수익 포기

### 5.2 기대값 비교 (가상 시나리오 100거래)

**전략 A: 고정 TP +7%**
- 40건 × -2% = -80%
- 60건 × +7% = +420%
- 순수익 = +340%, 거래당 +3.4%

**전략 B: Trailing-only -3% (TP 없음)**
- 40건 × -2% = -80% (손절은 동일)
- 25건 × +3% = +75% (TP 없어도 소폭 이익 후 trailing 발동)
- 15건 × +8% = +120%
- 10건 × +15% = +150%
- 10건 × +27% = +270% (상한가 근접)
- 순수익 = +535%, 거래당 +5.35%

**전략 C: Hybrid (50% at +7%, 50% trailing)**
- 40건 × -2% = -80%
- 30건 × +7% = +210% (전량 TP, +7% 도달 후 하락)
- 30건 × (3.5% + 7.5%) = +330% (50% TP + 50% trailing)
- 순수익 = +460%, 거래당 +4.6%

> 주의: 위 시나리오는 현재 시스템의 실적 데이터 기반 추정. 정확한 수치는 backtest_abc.py 확장 백테스트 필요.

### 5.3 핵심 결론

| 질문 | 답변 |
|------|------|
| 고정 TP 제거하면 기대값 증가? | **Yes**, 단 fat-tail이 충분히 두꺼운 경우 (테마주 해당) |
| 승률은? | 하락 (60% → 45% 예상) |
| 심리적 부담? | 증가 (미실현 이익 반납 경험 빈번) |
| round-trip 리스크? | stepped trailing으로 완화 가능 |

## 6. 실행 권장안 (Actionable Recommendations)

### 6.1 1순위 권장: Stepped Trailing + 고정 TP 제거

**구현 변경:**

```
현재: should_sell() → TP +7% 먼저 체크 → trailing -3% 나중에
변경: should_sell() → TP 제거 → stepped trailing만 적용
```

**Stepped Trailing 파라미터:**

| 조건 | 동작 |
|------|------|
| pnl < 0% | 고정 SL -2% 유지 (변경 없음) |
| pnl >= +5% | stop을 0% (breakeven)으로 이동 |
| pnl >= +10% | stop을 +5%로 이동 |
| pnl >= +15% | stop을 +10%로 이동 |
| pnl >= +20% | stop을 peak - 3%로 전환 (기존 trailing) |

**예상 효과:**
- 상한가 종목: +7% → +27% (3.86배 수익 증가)
- 중간 상승 종목 (+8~10%): +7% → +5~7% (소폭 감소 가능)
- 손절 종목: 변화 없음 (-2%)
- **round-trip 방지:** +5% 이상 도달 후 원금 손실 불가

### 6.2 2순위 권장: 분할 매도 Hybrid

고정 TP 완전 제거가 심리적으로 부담될 경우:

| 단계 | 수량 | 조건 |
|------|------|------|
| 1차 | 50% | +7% 도달 |
| 2차 | 50% | stepped trailing |

**구현 고려사항:**
- position_db에 `partial_sold` 필드 추가 필요
- 주문 수량 관리 복잡도 증가
- KIS API 부분 매도 주문 가능 여부 확인 필요

### 6.3 3순위: 시간 기반 Trailing 조정

장마감 접근 시 trailing을 좁히는 방식:

```python
def get_dynamic_trailing(base_pct: float, hour: int) -> float:
    if hour < 10:
        return base_pct * 1.5   # -4.5% (넓게)
    elif hour < 14:
        return base_pct         # -3.0% (기본)
    else:
        return base_pct * 0.67  # -2.0% (좁게)
```

### 8.3 Stepped의 리스크 — "되돌림" 시나리오

코오롱티슈진과 큐렉소에서 Stepped가 현재보다 불리:

| 종목 | 현재(+7% TP) | Stepped | 차이 |
|------|------------|---------|------|
| 코오롱티슈진 | +7% | +4.7% | **-2.3%** |
| 큐렉소 | +7% | +0.5% | **-6.5%** |

+7%를 넘었다가 되돌아온 종목에서 손해. 하지만 **상한가 종목의 추가 수익(+33.6%)이 되돌림 손실(-8.8%)을 크게 상쇄.**

### 8.4 핵심 질문: 수익이 +7% 도달 후 어떻게 분포하는가?

오늘 +7% 이상 도달한 종목 6개:

| 종목 | 장중 최고 | 종가 | +7% 후 추가 상승 | 패턴 |
|------|---------|------|----------------|------|
| 에스폴리텍 | +20.3% | +29.99% | **+22.99%p 추가** | 상한가 |
| 대영포장 | +27.3% | +29.98% | **+22.98%p 추가** | 상한가 |
| 코오롱티슈진 | +22.4% | +17.1% | 고점에서 +15.4%p 추가, 종가는 +10.1%p | 되돌림 |
| 큐렉소 | +22.9% | +15.1% | 고점에서 +15.9%p 추가, 종가는 +8.1%p | 되돌림 |
| 셀바스AI | +27.3% | +23.8% | **+20.3%p 추가** | 오후 재상승 |
| 아주IB투자 | +15.7% | +4.7% | 고점에서 +8.7%p 추가, 종가는 -2.3%p | 큰 되돌림 |

**6종목 중 3종목이 +7% 후에도 +20%p 이상 추가 상승 (50%).** 나머지 3종목도 종가가 +7% 이상 (되돌려도 +4.7~17.1%).

→ Stepped trailing에서 **+7% 도달 후 breakeven(0%) 아래로 떨어진 종목 = 0개**. round-trip 리스크가 오늘 데이터에서는 발현되지 않음.
