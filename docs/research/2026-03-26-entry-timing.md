## 1. 갭상승 종목의 일중 패턴

### 1.1 갭 크기별 행동 패턴

| 갭 크기 | 전형적 패턴 | 갭 충전(fill) 확률 |
|---------|-----------|-----------------|
| 1~2% | 높은 갭 충전율 (~45%) | 비교적 쉽게 되돌림 |
| 2~5% | 갭 충전율 30~33% | 혼재 (뉴스 유무에 따라) |
| 5~10% | 갭 충전율 ~10% | 대부분 Gap-and-Go, 일부 고점 후 되돌림 |
| 10%+ | 갭 충전율 <10% | 대부분 유지하나 장초반 고점 후 조정 빈번 |
| 20%+ | 극소수만 갭 충전 | 동시호가 과열 → 09:00~09:30 급등 → 이후 급격한 이익실현 |

### 1.2 한국 시장 시간대별 특성

| 시간대 | 특성 | 매매 의미 |
|--------|------|---------|
| 08:30~09:00 | 동시호가, 가격 발견 | 갭 크기 확정 |
| **09:00~09:30** | **최대 변동성 구간**, 동시호가 물량 소화 | 장초반 고점/저점 형성 |
| 09:30~10:00 | Opening Range 확정, 초기 방향성 판단 | **1차 되돌림 시작** |
| 10:00~11:00 | 기관 매매 본격화, 추세 안정화 | **최적 진입 윈도우** |
| 11:00~12:30 | 점심 시간대, 거래량 감소 | 낮은 변동성, 소폭 되돌림 |
| 13:00~14:30 | 오후 추세 재개 | 방향성 확인 구간 |
| 14:30~15:30 | 장마감 수급, 종가 매매 | 일중 모멘텀 마감 |

### 1.3 핵심 통계

- **갭상승 종목의 장초반 고점**: 09:00~09:30에 당일 고점을 찍는 경우가 가장 빈번 (특히 10%+ 갭상승)
- **일중 모멘텀 효과**: KAIST 연구에 따르면, 09:00~09:30과 14:30~15:00에 일중 모멘텀이 통계적으로 유의
- **삼성증권 갭 분석**: 7%+ 갭상승 종목은 10분 관망 후 판단, 10시까지 상한가 진행 여부로 갈림

## 2. 풀백(되돌림) 진입 전략

### 2.1 전략 A: 시간 기반 지연 진입 (Time-Delayed Entry)

**원리:** 장초반 과열 매수세 소화를 기다린 후 진입

```
규칙:
- 시그널 감지 시 즉시 매수하지 않고 대기 목록에 등록
- 10:00 이후부터 매수 가능 (최소 장 시작 후 60분 대기)
- 갭 크기 20%+: 10:30 이후부터 매수 가능
- 갭 크기 10~20%: 10:00 이후부터 매수 가능
- 갭 크기 5~10%: 09:45 이후부터 매수 가능
```

**장점:** 구현 단순, 장초반 고점 매수 회피
**단점:** 강한 모멘텀(상한가 직행) 종목 놓칠 수 있음

**2026-03-26 시뮬레이션:**
- 에코플라스틱: 09:30 고점 4,360 → 10:00경 되돌림 후 재진입 시 손절 회피 가능성 높음
- 알테오젠: 09:30 고점 393,000 → 10:00 이후 진입 시 더 낮은 가격 확보

### 2.2 전략 B: 가격 기반 풀백 진입 (Pullback Entry)

**원리:** 일중 고점 대비 일정 % 하락 시 지정가 매수

```
규칙:
- 시그널 감지 시 현재가를 "기준가"로 기록
- 기준가 대비 -1.5%~-3% 아래에 지정가 매수 주문
- 갭 크기별 차등:
  - 갭 5~10%: 기준가 -1.5% 지정가
  - 갭 10~20%: 기준가 -2.0% 지정가
  - 갭 20%+: 기준가 -3.0% 지정가
- 제한 시간: 14:00까지 미체결 시 주문 취소
```

**장점:** 장초반 고점 회피, 더 좋은 가격에 진입
**단점:** 되돌림 없이 상승 지속 시 진입 실패

**2026-03-26 시뮬레이션:**
- 에코플라스틱(+24.7% 갭): 4,360 × 0.97 = 4,229 지정가 → 체결 가능(종가 4,240), 손절 미도달
- 알테오젠(+9.6% 갭): 393,000 × 0.98 = 385,140 지정가 → 체결 가능, 종가 381,000은 손절 근접

### 2.3 전략 C: VWAP 기반 진입

**원리:** VWAP(거래량가중평균가격) 아래로 내려올 때 매수

```
규칙:
- 시그널 감지 후 VWAP 실시간 계산
- 현재가가 VWAP 터치 또는 하회 시 매수
- 추가 조건: VWAP 터치 후 반등 확인 (1분봉 양봉 확인)
- VWAP 이하 장기 체류 (30분+) 시 매수 포기 (모멘텀 소멸)
```

**장점:** 기관 매매 평균가 기준으로 합리적 진입
**단점:** VWAP 실시간 계산 필요 (분봉 데이터 수집 필수), 구현 복잡도 높음

### 2.4 전략 D: Opening Range Breakout (ORB) 응용

**원리:** 처음 15~30분의 가격 범위를 기준으로 진입

```
규칙:
- 09:00~09:30 (또는 09:15) 동안의 고가/저가를 Opening Range로 설정
- Opening Range 하단까지 되돌림 후 반등 시 매수
- Opening Range 하단 이탈 시 매수 포기
- Stop-loss: Opening Range 하단 아래
```

**장점:** 명확한 가격 기준, 손절 기준 명확
**단점:** Opening Range 하단까지 되돌림이 안 올 수 있음

## 3. 거래량-가격 확인 (Volume-Price Confirmation)

### 3.1 건강한 풀백 vs 모멘텀 소멸

| 구분 | 건강한 풀백 (재진입 가능) | 모멘텀 소멸 (진입 금지) |
|------|----------------------|---------------------|
| 풀백 거래량 | 랠리 대비 **감소** (50% 이하) | 랠리 대비 **유지 또는 증가** |
| 풀백 속도 | 느린 하락 (계단식) | 급격한 하락 (수직) |
| 풀백 깊이 | 상승폭의 38~50% 되돌림 | 상승폭의 61.8%+ 되돌림 |
| 호가 특성 | 매수 잔량 유지 | 매도 잔량 급증 |

### 3.2 구현 가능한 거래량 필터

```
확인 조건 (모두 충족 시 매수):
1. 풀백 구간 평균 거래량 < 랠리 구간 평균 거래량의 60%
2. 현재가가 갭상승 시초가 이상 유지
3. 풀백 깊이가 장초반 상승폭의 50% 이내
```

### 3.3 실용적 단순화

현재 시스템에서 분봉 거래량 데이터를 실시간 수집하기 어려운 경우:

```
간단 대안:
- 매수 시점의 체결강도(매수비율) 확인
- 체결강도 50% 이상일 때만 매수 (매수세 우위 확인)
- KIS API에서 체결강도 조회 가능
```

## 4. 시장 레짐 필터 (Market Regime Filter)

### 4.1 KOSPI 지수 기반 필터

```
규칙:
- KOSPI 당일 등락률 확인 (09:30 기준)
- KOSPI < -1.0%: 매수 수량 50% 축소
- KOSPI < -2.0%: 매수 중단
- KOSPI > +0.5%: 정상 매수
```

**근거:** 시장 전체가 약세일 때 개별 종목의 갭상승도 되돌림 확률이 높음

### 4.2 상대강도 필터

```
규칙:
- 종목의 갭상승률 vs KOSPI/KOSDAQ 지수 등락률 비교
- (종목 갭상승률) - (지수 등락률) = 상대강도
- 상대강도 5%p 이상인 종목만 매수 대상
- 시장 하락 중 갭상승 종목 = 강한 개별 재료 → 상대적으로 안전
```

### 4.3 당일 누적 손실 기반 (현재 구현됨)

현재 시스템의 `MAX_DAILY_LOSS_PCT` 체크는 유지하되, 추가 필터:

```
- 당일 손절 2회 연속 발생 시 → 남은 매수 대상 재검증 (시간 지연 추가)
- 당일 손절 3회 발생 시 → 당일 매수 완전 중단
```

## 5. 학술/정량 리서치 요약

### 5.1 갭 충전(Gap Fill) 통계

- **전체 갭**: 약 76%가 당일 내 갭 충전 (평균 통계)
- **큰 갭 (5%+)**: 갭 충전율 약 9.6%로 급감 → Gap-and-Go 우세
- **작은 갭 (1~2%)**: 갭 충전율 ~45% → 평균회귀 우세
- 결론: **큰 갭은 추세추종, 작은 갭은 평균회귀**가 유리

### 5.2 학술 연구 핵심 발견

1. **모멘텀과 평균회귀 공존**: 같은 시장에서 두 전략이 동시에 수익 가능 (regime-dependent)
2. **Fade-the-Gap 포트폴리오**: Fama-French 3-factor 보정 후에도 양(+)의 유의한 알파
3. **갭 크기가 전략 결정**: 큰 갭 = 모멘텀 (하지만 장초반 진입은 위험), 작은 갭 = 평균회귀
4. **한국 시장 일중 모멘텀**: 장초반과 장후반에 통계적으로 유의한 모멘텀 효과

### 5.3 2026-03-26 데이터에 적용

4종목 모두 갭 5%+ → Gap-and-Go 범주이나, **장초반 고점 매수는 모멘텀 전략이 아니라 추격매수**. 모멘텀 전략의 올바른 진입은 풀백 후 재상승 확인 시점.

## 6. 실전 구현 권장안

# 진입 로직:

# 3. 지연 시간 경과 후 지정가 주문 (기준가 × discount)

**이것은 4종목(Type 4)에서만 검증한 편향된 결론이었음.** 19종목 전체로 확대하면:

**지정가 매수 시 놓치는 종목:**
- Type 1 (상한가 직행): 에스폴리텍, 대영포장 → 지정가가 체결되기 전에 상한가 도달. **+29.99% 수익 기회 상실**

**지정가 매수가 유효한 종목:**

**핵심 딜레마:**
```
시장가 즉시 매수: Type 1, 3에서 수익 / Type 4에서 손절
지정가 풀백 매수: Type 4에서 개선 / Type 1에서 기회 상실
```

### 10.4 수정된 결론

기존 연구의 "지정가 매수 전환이 최우선"이라는 결론은 **편향된 표본(손절 4종목)에 기반한 오류**였음.

19종목 전체로 보면:

> **1. 현재 시스템(시장가 즉시 + -2% 손절)이 생각보다 나쁘지 않다.**
> 상한가 직행 종목의 대형 수익이 손절 종목의 소형 손실을 상쇄한다.
> 5종목 매수 시: +41.6% (시장가) vs +8.4% (지정가) → 시장가가 **5배** 유리.
>
> **2. 진짜 문제는 "손절당한 종목"이 아니라 "매수하지 못한 종목"일 수 있다.**
> 오후 상한가 4종목(세우글로벌, 한국ANKOR유전, 디에이치엑스컴퍼니, 셀바스AI)을 시스템이 포착하지 못했다면, 그 기회비용이 더 크다.
>
> **3. -2% 손절의 역할 재평가.**
> 손절당한 종목(Type 4) 4개 × -2% = -8%. 하지만 상한가 종목 2개만 잡아도 +47.6%. 순수익 +39.6%. 손절은 "보험료"이며, 상한가 종목 1개가 손절 20회분을 상쇄한다.
>
> **4. 개선 방향 재설정.**
> ~~손절폭 조정~~ → 효과 제한적
> ~~지정가 매수 전환~~ → 상한가 종목 놓침
> **→ 매수 종목 선별력 강화 + 오후 급등 종목 포착 능력이 핵심**
