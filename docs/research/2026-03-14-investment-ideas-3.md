## 아이디어 11: 시스템 성과 대시보드 (백테스트 메타 분석)

### 개요
두 프로젝트(theme_analysis, signal_analysis)의 **분석 시스템 자체의 성과**를 추적하고 평가하는 메타 분석 시스템. "이 시스템의 신호를 따라갔으면 실제로 돈을 벌었을까?"를 객관적 데이터로 검증.

### 핵심 가치
- **분석 깊이**: 시스템 신호의 정확도/수익률을 장기적으로 추적
- **편의성**: 어떤 신호 소스(Vision/KIS/Combined/테마)를 더 신뢰해야 하는지 한눈에 파악
- **실시간성**: 매일 성과 업데이트 + 성과 급변 시 알림

### 문제 의식

현재 두 프로젝트 모두 시뮬레이션을 제공하지만, **시스템 전체의 신뢰도를 종합 평가**하는 메타 레벨 분석은 부재:

- Vision 분석은 언제 잘 맞고 언제 틀리나?
- Combined가 Vision 단독보다 정말 나은가?
- 테마 예측의 적중률은 시간이 지나면서 개선되나?
- 시장 국면(상승장/하락장/횡보장)별로 어떤 신호가 더 유효한가?

### 분석 항목

#### 1. 신호 소스별 성과 비교
```
┌──────────────────────────────────────────────────┐
│ 📊 30일 누적 성과 비교                               │
├──────────────┬────────┬────────┬────────┬────────┤
│ 지표          │ Vision │ KIS    │Combined│ 테마예측│
├──────────────┼────────┼────────┼────────┼────────┤
│ 적극매수 적중률│ 68.2%  │ 72.1%  │ 78.5%  │ -      │
│ 평균 수익률   │ +3.2%  │ +2.8%  │ +4.1%  │ -      │
│ 최대 손실     │ -8.3%  │ -5.1%  │ -4.2%  │ -      │
│ 샤프 비율     │ 1.2    │ 1.5    │ 1.8    │ -      │
│ 테마 적중률   │ -      │ -      │ -      │ 65%    │
│ 대장주 적중률  │ -      │ -      │ -      │ 58%    │
└──────────────┴────────┴────────┴────────┴────────┘
```

#### 2. 시장 국면별 성과
```
시장 국면 판정 기준:
- 상승장: KOSPI 20일 MA 위 + F&G > 50
- 하락장: KOSPI 20일 MA 아래 + F&G < 40
- 횡보장: 기타

┌─────────┬─────────────┬─────────────┬─────────────┐
│ 국면     │ Vision 적중률│Combined 적중률│ 테마 적중률  │
├─────────┼─────────────┼─────────────┼─────────────┤
│ 상승장   │ 75.3%       │ 82.1%       │ 72.0%       │
│ 하락장   │ 45.2%       │ 55.8%       │ 38.5%       │
│ 횡보장   │ 58.6%       │ 64.3%       │ 51.2%       │
└─────────┴─────────────┴─────────────┴─────────────┘

→ 인사이트: "하락장에서는 Combined도 신뢰도가 낮음.
   현재 시장이 하락 국면이라면 매수 신호를 보수적으로 해석할 것"
```

#### 3. 신호 강도별 성과
```
적극매수 vs 매수 — 정말 차이가 있나?

적극매수 (비전 점수 80+):
  적중률 78%, 평균 수익 +4.8%, 표본 42건

매수 (비전 점수 60~79):
  적중률 62%, 평균 수익 +2.1%, 표본 85건

→ 인사이트: "적극매수 신호에 집중하면 승률과 수익률 모두 향상"
```

#### 4. 시간 경과에 따른 성과 추이
```
월별 적중률 추이 (시스템이 개선되고 있는가?)
1월: 62% → 2월: 67% → 3월: 72% (↑ 개선 중)

→ 인사이트: "프롬프트/로직 개선이 실제 성과 향상으로 이어지고 있음"
```

### 텔레그램 알림 예시
```
📊 [시스템 성과] 주간 리포트
━━━━━━━━━━━━━━━━━

이번 주 성과 (3/10~3/14):

• 평균 수익: +3.8%
• 최고: 한화에어로 +8.5%

⚠️ 주의: Vision 단독 신호 적중률이 2주 연속
   하락 중 (72%→65%). 프롬프트 검토 필요할 수 있음.

📌 현재 가장 신뢰할 수 있는 소스: Combined
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| simulation 결과 (두 프로젝트) | 이미 계산된 수익률 데이터 |
| 매매 신호 히스토리 (30일) | signal_analysis → 일별 신호 변동 추적 |
| 테마 예측 히스토리 (20일) | theme_analysis → forecast-history/ |
| 종목 시세 히스토리 (90일) | theme_analysis → history/ |
| macro 데이터 | 시장 국면 판정 (F&G, VIX, 지수 MA) |

### 기술 구현
- 히스토리 JSON 파싱 → 신호 발생일 기준 D+1, D+3, D+5 수익률 산출
- 시장 국면 분류 → macro 데이터 기반 자동 판정
- 통계 계산 → 적중률, 평균수익, 샤프비율, 최대손실 등
- 웹 대시보드 + 주간 텔레그램 리포트
- GitHub Actions: 매일 장 마감 후 성과 업데이트

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (히스토리 데이터 통계 분석, 신규 수집 불필요)
- **기대효과**: 시스템에 대한 객관적 신뢰 근거 확보. "어떤 상황에서 어떤 신호를 믿어야 하는가"에 대한 데이터 기반 답변. 시스템 개선 방향 도출.

## 아이디어 12: 섹터/업종 자금 흐름 맵

### 개요
개별 종목이 아닌 **섹터/업종 단위**로 자금의 유입/유출 흐름을 시각화하는 시스템. 현재 어떤 섹터로 돈이 몰리고 있고, 어떤 섹터에서 빠지고 있는지를 **히트맵**으로 한눈에 파악.

### 핵심 가치
- **편의성**: 개별 종목 수백 개를 볼 필요 없이 "큰 그림" 파악
- **분석 깊이**: 섹터 간 자금 이동 패턴 = 시장의 방향성 판단 근거
- **실시간성**: 장중 업데이트 + 급격한 자금 이동 시 알림

### 핵심 시각화

#### 섹터 히트맵 (트리맵)
```
┌──────────────────────────────────────────────┐
│                                              │
│   ██████████████   ████████   ████████       │
│   ██ 2차전지  ██   ██ AI  ██   ██자동차██     │
│   ██ +850억  ██   ██반도체██   ██+220억██     │
│   ██ (+3.2%)██   ██+540억██   ██(+1.8%)██    │
│   ██████████████   ██(+2.8%)█   ████████     │
│                    ████████                   │
│   ████████   ██████   ███   ███              │
│   ██바이오██   ██건설██  ██엔터██ ██금융██      │
│   ██-320억██   ██-180억██ ██-95억██ ██+50억██  │
│   ██(-2.1%)██  ██(-1.5%)██(-1.2%)██(+0.3%)██  │
│   ████████   ██████   ███   ███              │
│                                              │
│ 🟢 유입 (+)  🔴 유출 (-)  크기 = 거래대금      │
└──────────────────────────────────────────────┘
```

#### 섹터 자금 흐름 시계열
```
최근 5일 섹터별 외국인 순매수 추이 (억원):

섹터       │ 3/10  3/11  3/12  3/13  3/14 │ 5일합계
──────────┼──────────────────────────────┼────────
2차전지   │ +120  +180  +250  +150  +150 │ +850
AI반도체  │ +80   +95   +120  +130  +115 │ +540
자동차    │ -30   +50   +80   +60   +60  │ +220
바이오    │ +40   +20   -50   -180  -150 │ -320
건설      │ -20   -30   -40   -50   -40  │ -180
```

### 섹터 분류 방법

두 프로젝트의 기존 데이터를 활용한 자동 분류:

```
1차: 테마 기반 분류 (theme_analysis)
  - AI 테마 분석 결과에서 테마명 → 섹터 매핑
  - 예: "2차전지 소재", "2차전지 장비" → 2차전지 섹터

2차: 종목코드 기반 분류
  - KRX 업종 분류 데이터 활용 (pykrx로 조회 가능)
  - KOSPI/KOSDAQ 업종 코드 매핑

3차: AI 보조 분류
  - 테마에도 업종에도 깔끔하게 안 맞는 종목은 Gemini로 분류
```

### 텔레그램 알림 예시
```
🗺️ [자금 흐름] 섹터 이동 감지
━━━━━━━━━━━━━━━━━

📈 자금 유입 가속 (3일 연속 증가)
• 2차전지: +850억 (5일) — 외국인 주도
• AI반도체: +540억 (5일) — 기관+외국인

📉 자금 유출 전환 (매수→매도)
• 바이오: -330억 (3일 연속 유출)
  → 직전 5일 +200억 유입에서 반전

🔄 로테이션 감지
"바이오 → 2차전지 자금 이동 패턴.
 바이오 대장주 하락 + 2차전지 거래대금 급증.
 섹터 로테이션 초기 단계로 판단."
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| AI 테마 분석 결과 | theme_analysis → 종목↔테마 매핑 (섹터 분류 근거) |
| 투자자별 매매동향 | theme_analysis → 종목별 외국인/기관 순매수 |
| 거래대금 TOP30 | theme_analysis → 섹터별 거래 규모 산출 |
| 등락률 데이터 | theme_analysis → 섹터별 평균 수익률 |
| 히스토리 (90일) | theme_analysis → 섹터 시계열 구축 |
| pykrx | 이미 의존성에 포함 → 업종 분류 데이터 조회 가능 |

### 기술 구현
- **섹터 분류**: pykrx 업종코드 + theme_analysis 테마 매핑 하이브리드
- **집계 엔진**: 종목별 수급/거래량/등락률 → 섹터 단위 합산
- **히트맵 웹**: React + D3.js 트리맵 또는 기존 TailwindCSS 기반 그리드
- **로테이션 감지**: 섹터 간 자금 순유입 역전 패턴 탐지
- **알림**: 로테이션 감지 / 급격한 유출입 시 텔레그램 전송
- **GitHub Actions**: 장중 2~3회 데이터 갱신

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (종목→섹터 매핑 + 집계 로직 + 시각화)
- **기대효과**: 개별 종목에 매몰되지 않고 시장의 큰 흐름을 파악. "돈이 어디로 가고 있는가"에 대한 직관적 답변. 섹터 로테이션 타이밍 포착.

## 아이디어 13: 종목 유사 패턴 매칭 (과거 차트 유사도)

### 개요
특정 종목의 현재 차트 패턴과 **과거에 유사했던 차트 패턴**을 찾아, 이후 주가가 어떻게 움직였는지를 보여주는 시스템. "지금과 비슷했던 과거 사례에서는 어떤 일이 벌어졌나?"에 대한 통계적 답변.

### 핵심 가치
- **분석 깊이**: 단순 기술적 지표가 아닌 차트 형태(shape) 기반 분석
- **편의성**: "이 종목 앞으로 어떻게 될까?"에 대한 과거 사례 기반 답변
- **실시간성**: 매일 갱신된 패턴 매칭 결과 제공

### 작동 원리

```
현재 종목의 최근 N일 가격 패턴
           ↓
정규화 (시작점 = 0%, 상대 변화율로 변환)
           ↓
과거 히스토리에서 유사 패턴 검색
(DTW - Dynamic Time Warping 또는 코사인 유사도)
           ↓
유사도 상위 K개 사례 추출
           ↓
각 사례의 "이후 N일" 수익률 통계
           ↓
예상 시나리오 제시
```

### 분석 예시
```
📊 삼성SDI (006400) 패턴 매칭 결과
━━━━━━━━━━━━━━━━━

현재 패턴: 최근 20일간 +12% 상승 후 3일 횡보
유사 패턴: 5건 발견 (유사도 85%+)

┌──────────────────────────────────────┐
│  유사 사례         │ 이후 5일│이후 10일│
├──────────────────────────────────────┤
│ 1. 삼성SDI 2025-11 │ +4.2%  │ +7.8%  │
│ 2. LG에너지 2025-09│ +2.1%  │ +5.3%  │
│ 3. 삼성SDI 2025-06 │ -1.5%  │ +3.2%  │
│ 4. SK이노 2025-12  │ +5.8%  │ +9.1%  │
│ 5. 에코프로 2026-01│ -3.2%  │ -5.8%  │
├──────────────────────────────────────┤
│ 평균              │ +1.5%  │ +3.9%  │
│ 상승 확률          │ 60%    │ 80%    │
└──────────────────────────────────────┘

💡 해석: 유사 패턴 5건 중 4건에서 10일 후
   상승. 다만 에코프로 사례처럼 횡보 후 하락
   전환 가능성도 20% 존재. 현재 수급(외국인
   연속 매수)이 긍정적 차별 요인.
```

### 매칭 범위

| 매칭 유형 | 설명 | 활용 |
|-----------|------|------|
| **자기 매칭** | 같은 종목의 과거 패턴 | "이 종목이 이전에 비슷한 움직임을 보인 적은?" |
| **동일 섹터 매칭** | 같은 테마/섹터 종목의 과거 패턴 | "2차전지 종목들의 유사 패턴 결과는?" |
| **전체 매칭** | 모든 종목의 과거 패턴 | "시장 전체에서 유사 패턴 사례는?" |

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 종목 시세 히스토리 (90일) | theme_analysis → history/ (패턴 매칭 소스) |
| 분봉 데이터 | theme_analysis → intraday-history (단기 패턴 매칭) |
| 거래량 프로파일 | theme_analysis → volume-profile (가격+거래량 복합 패턴) |
| 테마 분류 | theme_analysis → AI 테마 분석 (동일 섹터 매칭 범위 설정) |
| 매매 신호 | signal_analysis → 현재 종목 상태 결합 |

### 기술 구현
- **패턴 정규화**: 가격 시계열을 상대 변화율로 정규화
- **유사도 계산**: numpy/scipy의 DTW 또는 코사인 유사도 (라이브러리 활용)
- **검색 최적화**: 90일 × 200종목 = ~18,000 구간 → 사전 인덱싱으로 실시간 가능
- **결과 시각화**: 현재 패턴 vs 유사 패턴 오버레이 차트
- **텔레그램**: 적극매수 종목에 대해 패턴 매칭 결과 자동 첨부
- **GitHub Actions**: 일 1회 전체 종목 패턴 매칭 갱신

### 한계 & 보완
- **한계**: 90일 히스토리로는 표본이 제한적 → 장기적으로 히스토리 확장 필요
- **보완**: pykrx로 과거 수년 데이터 추가 수집 가능 (1회 작업)
- **주의**: 과거 패턴이 미래를 보장하지 않음 → 결과에 항상 신뢰구간/표본수 명시

### 난이도 & 기대효과

## 아이디어 14: 이상 거래 탐지 & 알림

### 개요
장중 **비정상적인 거래 패턴**(거래량 폭증, 호가 이상, 가격 급변 등)을 실시간으로 탐지하여 즉시 알리는 시스템. 뉴스보다 빠르게 "무언가 일어나고 있다"는 것을 포착.

### 핵심 가치
- **실시간성**: 이상 거래는 속도가 생명 — 뉴스/공시 전에 포착
- **분석 깊이**: 단순 가격 변동이 아닌 복합적 이상 패턴 분석
- **편의성**: 100+ 종목을 감시할 필요 없이 이상 발생 시만 알림

### 탐지 패턴

| 패턴 | 탐지 기준 | 의미 |
|------|----------|------|
| **거래량 폭발** | 5분봉 거래량 > 20일 평균의 500% | 대형 이벤트/재료 발생 가능 |
| **가격 급등락** | 5분 내 ±3% 이상 변동 | 급등 시 진입 기회 또는 급락 시 탈출 필요 |
| **상한가/하한가 접근** | ±25% 이상 또는 ±20% 이상 도달 | 극단적 수급 불균형 |
| **거래량 없는 급등** | 가격 +2% 이상 but 거래량 평소 이하 | 호가 조작 또는 매도 물량 고갈 |
| **수급 급변** | 분봉 단위 외국인/기관 대량 매수/매도 | 기관 수급 전환 신호 |
| **갭 이상** | 시가 vs 전일 종가 괴리 ±5% 이상 | 장전 재료 반영, 갭 매매 기회 |
| **동시 급등** | 같은 테마 3종목+ 동시에 +2% 이상 | 테마 전체 급등 시작 |

### 텔레그램 알림 예시
```

💡 판단: 거래량 + 가격 동시 급증.
   방산 테마 내 다른 종목(LIG넥스원 +2.1%)도
   동반 상승 중. 테마 전체 모멘텀 발생 가능성.

```

- **뉴스 역추적**: 이상 감지 시 Naver News API로 관련 뉴스 자동 검색
- **알림 쿨다운**: 같은 종목 30분 이내 재알림 방지

- **난이도**: ★★★☆☆ (기존 분봉 수집 인프라 위에 탐지 로직 추가)
- **기대효과**: 뉴스/공시보다 빠르게 "뭔가 일어나고 있다"를 포착. 단타/스윙 모두에서 진입 타이밍 확보. 동시 급등 감지로 테마 초기 모멘텀 포착.

## 아이디어 15: 투자 시나리오 시뮬레이터

### 개요
"만약 ~했다면?" 질문에 답하는 시뮬레이션 도구. 다양한 투자 전략을 **과거 데이터에 적용**하여 성과를 비교하고, 최적 전략을 찾는 what-if 분석기.

### 핵심 가치
- **분석 깊이**: 전략 간 성과를 객관적 수치로 비교
- **편의성**: 복잡한 코딩 없이 전략을 텍스트로 정의하면 자동 시뮬레이션
- **실시간성**: 새로운 전략 아이디어를 즉시 테스트하고 결과 확인

### 시뮬레이션 시나리오 예시

#### 시나리오 1: 신호 소스 비교
```
👤 "Vision 적극매수만 따라가기 vs Combined 적극매수만 따라가기,
    최근 30일 수익률 비교해줘"

🤖 시뮬레이션 결과 (30일, 1주일 보유 기준):
   ┌─────────────┬──────────┬──────────┐
   │             │ Vision만  │Combined만│
   ├─────────────┼──────────┼──────────┤
   │ 총 매매 횟수 │ 42건     │ 18건     │
   │ 승률        │ 64.3%   │ 77.8%    │
   │ 평균 수익   │ +2.1%   │ +3.8%    │
   │ 최대 수익   │ +12.5%  │ +9.8%    │
   │ 최대 손실   │ -8.3%   │ -4.2%    │
   │ 누적 수익률  │ +18.5%  │ +22.1%   │
   │ 샤프 비율   │ 1.2     │ 1.9      │
   └─────────────┴──────────┴──────────┘
   → Combined이 매매 횟수는 적지만 승률/샤프 모두 우수
```

#### 시나리오 2: 진입 조건 조합 테스트
```
👤 "적극매수 + 외국인 3일 순매수 + 테마 대장주 — 이 3조건
    모두 충족 시 매수하면 어떻게 됐을까?"

🤖 시뮬레이션 결과 (90일):
   • 조건 충족 횟수: 12건
   • 승률: 91.7% (11승 1패)
   • 평균 수익 (5일 보유): +5.8%
   • 평균 수익 (10일 보유): +8.2%
   • 유일한 손실: 에코프로비엠 -2.1% (공매도 급증)

   → 3중 필터 전략은 기회는 적지만 승률이 매우 높음
```

#### 시나리오 3: 보유 기간 최적화
```
👤 "적극매수 신호 후 며칠 보유가 최적인지 분석해줘"

🤖 보유 기간별 수익률 분석 (Combined 적극매수, 90일):
   ┌──────────┬──────────┬──────────┐
   │ 보유 기간 │ 평균수익  │ 승률     │
   ├──────────┼──────────┼──────────┤
   │ 1일      │ +1.2%   │ 62%      │
   │ 3일      │ +2.8%   │ 70%      │
   │ 5일      │ +3.8%   │ 75%      │ ← 수익/리스크 최적
   │ 10일     │ +4.1%   │ 72%      │
   │ 20일     │ +3.5%   │ 68%      │
   └──────────┴──────────┴──────────┘
   → 5일 보유가 수익률 대비 승률 최적 지점
   → 10일 이후 수익률 정체 + 승률 하락 (평균 회귀)
```

#### 시나리오 4: 손절/익절 최적화
```
👤 "손절 -3% vs -5% vs 없음, 어떤 게 나아?"

🤖 손절 기준별 성과 (Combined 적극매수, 5일 보유, 90일):
   ┌──────────┬──────────┬──────────┬──────────┐
   │          │ -3% 손절 │ -5% 손절 │ 손절 없음 │
   ├──────────┼──────────┼──────────┼──────────┤
   │ 평균수익  │ +2.5%   │ +3.2%   │ +3.8%   │
   │ 최대손실  │ -3.0%   │ -5.0%   │ -8.3%   │
   │ 누적수익  │ +32%    │ +38%    │ +35%    │
   │ 샤프비율  │ 2.1     │ 2.0     │ 1.5     │
   └──────────┴──────────┴──────────┴──────────┘
   → -5% 손절이 누적수익 최대 + 최대손실 제한
   → -3% 손절은 너무 타이트해서 "정상 변동"에도 손절됨
```

- **이유**: 기존 분봉 수집 인프라(5분 주기)에 탐지 로직만 추가. 실시간성 최고. 특히 단타 매매에서 "뉴스보다 빠른 포착"은 결정적 이점. 동시 급등 감지로 테마 초기 모멘텀 포착도 가능.

### 3순위: 섹터 자금 흐름 맵 (#12)
- **이유**: 개별 종목에서 한 단계 올라간 "큰 그림" 제공. 종목 선택 전에 "어떤 섹터에 주목해야 하는가"를 먼저 결정할 수 있음.

### 4순위 이후
- 투자 시나리오 시뮬레이터 → 전략 최적화에 강력하나 백테스트 엔진 개발 필요
- 종목 유사 패턴 매칭 → 독창적이나 충분한 히스토리 데이터 확보 필요
