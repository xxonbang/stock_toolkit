# 모의투자 최적 a,b,c 백테스트 연구

## 개요
- **분석 날짜:** 2026-03-23
- **데이터:** stock-history.json 일봉 200일 (151종목)
- **변수:** a(익절%), b(손절%), c(급락 trailing stop%)

## 시나리오
1. 매집 후 수익률 >= a% → 익절
2. 매집 후 수익률 <= b% → 손절
3. 수익률 > 0% 이지만 고점 대비 c% 하락 → 즉시 매도
4. 장 마감(15:15) 시 강제 청산

## 탐색 범위
- a: 3, 5, 7, 10%
- b: -2, -3, -5, -7%
- c: -3, -5, -8%
- 총 48가지 조합

## 핵심 인사이트

1. **손절은 -2%가 최적** — 상위 6개 중 6개가 b=-2%
2. **익절은 높을수록 총수익 증가** — a=10%가 a=3%보다 3배
3. **c(trailing stop) -3%가 수익 보호에 효과적**
4. **승률 vs 수익률 트레이드오프** — 승률 73%(a=3)보다 승률 34%(a=10)가 총수익 3배

## 균형 조합 추천
- **공격적:** a=10%, b=-2%, c=-3% (최고 총수익, MDD 높음)
- **균형:** a=7%, b=-2%, c=-3% (총수익 93%, MDD 51%)
- **안정:** a=5%, b=-2%, c=-3% (총수익 85%, MDD 53%)

## 현재 설정과 비교

- 권장: a=7%, b=-2%, c=-3% → 총수익 +8,284%, 승률 36.6% (+46% 수익 개선)

## 위험 회피 + 수익 극대화 공식 연구

### 관찰된 패턴

1. **손절은 빠를수록 좋다** — b=-2%가 모든 시나리오에서 최적
2. **익절은 인내할수록 좋다** — a=10%가 a=3%보다 3배 수익
3. **trailing stop은 보유 기간이 길 때 효과적** — 당일에는 미미
4. **당일 청산 강제는 수익을 30~40% 깎음**

### 제안: 2단계 전략

**Phase 1 (당일):**
- 매수 직후 b=-2% 손절 즉시 적용
- 장중 a=7% 도달 시 즉시 익절
- 미도달 시 15:15 강제 청산

**Phase 2 (향후 개선):**
- 당일 수익률이 양수(+3% 이상)이면 익일까지 보유 허용
- 익일부터 c=-3% trailing stop 적용
- a=10% 도달 시 익절

### 현재 적용된 설정
- a=7%, b=-2%, c=-3% (균형 조합)
- 당일 15:15 강제 청산 유지
