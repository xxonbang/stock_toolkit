## 1. 기존 전략의 문제 진단

### 1.1 look-ahead bias 확인

기존 일봉 백테스트는 **당일 종가 시점의 거래량 데이터**로 종목을 선정하여 미래 정보를 참조했다.

| 구분 | 평균 수익 | 승률 |
|------|---------|------|
| 기존 백테스트 (일봉, look-ahead) | +13.8% | 95.7% |
| 분봉 기반 현실적 백테스트 | +0.56% | 37.8% |
| **실전 45건** | **-0.66%** | **31.1%** |

분봉 백테스트(+0.56%)가 실전(-0.66%)과 유사 → **분봉 백테스트만 신뢰 가능**.

### 1.2 종목 선정 로직의 근본 문제

**거래량비율(vol_rate) 정렬**은 평소 거래가 적은 소형주의 일시적 급등을 포착하지만, 시장 주도주와 무관한 종목을 선정한다.

2026-04-08 검증:
- 거래량비율 TOP2: 에스와이스틸텍 +4.9%, 대한해운 +1.1% → **합계 +6.0%**
- 거래대금 TOP2: 대우건설 +20.5%, 삼성전자우 +2.3% → **합계 +22.8%**

### 1.3 갭업 0~5% 상한의 문제

119종목(거래대금/거래량/등락률 TOP) 중 **54%(64종목)**가 등락률 ≥5%로 탈락. 당일 시장 주도주를 구조적으로 배제.

## 2. 기존 연구와의 교차분석

### 2.1 vs 04-04 전략비교 연구

04-04 연구에서 "현재 시스템이 Sharpe·PF 1위"라고 결론냈지만, 이 결론도 **look-ahead bias가 있는 백테스트** 기반이었다. 10개 전략 모두 같은 bias를 공유하므로 상대 비교는 유효하나, 절대 성과는 무의미.

**교차 결론:** 전략 간 상대 순위는 참고 가능. 그러나 절대 성과 수치(+0.95% 등)는 과대평가.

### 2.2 vs 04-06 거래대금 기준 연구

04-06 연구에서 이미 "거래대금 기반 필터"를 권장했으나, 당시에는 **최소 기준(500M~1B)으로만** 사용하고, 정렬 기준은 여전히 vol_rate에 의존했다.

**교차 결론:** 04-06의 "거래대금 최소 필터" 대신 **"거래대금 정렬"로 전환**해야 한다. 최소 필터는 하위 컷오프일 뿐이고, 정렬 기준이 핵심.

### 2.3 vs 04-06 거래량 정규화 연구

04-06 연구에서 제안한 방법들:
- `vol_ratio × log(prior_volume)` — 현재 사용 중이나 효과 미미
- `rank(vol_ratio) + rank(volume_KRW)` — 순위 복합 스코어
- 거래대금 직접 정렬 — **가장 단순하고 직관적**

**교차 결론:** 복합 스코어보다 **거래대금 직접 정렬이 우월**. 304일 검증에서 거래대금 TOP2(+3.14%)가 거래량비율 TOP2(+0.56%)를 압도.

### 2.4 vs 04-06 volume-momentum 연구

04-06 연구의 우선순위:
1. volume-rank API 도입 ✓ (이미 적용됨)
2. 전일 거래대금 최소 필터 → **정렬 기준으로 승격**
3. 복합 스코어 → **단순 거래대금 정렬로 대체**
4. 호가잔량 필터 → 추후 검토 가능

## 6. 주의사항 및 한계

1. **당일 거래대금 기준 백테스트는 약한 look-ahead bias를 포함**한다. 거래대금 순위가 장 초반에 결정되어 유지되는 경향이 있어 bias가 제한적이지만, 분봉 데이터(23일)에서의 검증과 교차 확인이 필요하다.

2. **분봉 데이터가 23거래일**로 표본이 작다. 분봉 데이터가 60거래일 이상 축적되면 재검증이 필요하다.

3. **동일종목 쿨다운**은 일봉 백테스트에서 검증되었으나, 실시간 거래대금 기반에서의 효과는 분봉 데이터 축적 후 재확인 필요.

4. **갭<15% 상한**도 일봉 기반 결과이며, 분봉 데이터에서는 10%~15% 구간의 표본이 적어 확정적이지 않다.
