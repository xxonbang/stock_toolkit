## 아이디어 26: 호가창 압력 분석기 (Order Book Pressure Analyzer)

### 개요
호가창의 **매수/매도 잔량 비율과 그 변화 속도**를 실시간으로 분석하여, 가격이 움직이기 **전에** 방향성을 예측하는 시스템. 대부분의 개인 투자자가 "가격이 움직인 후" 반응하는 반면, 호가창 압력은 가격 변동의 0.5~2초 전에 신호를 보낸다. 기관/알고리즘이 사용하는 마이크로스트럭처 분석을 개인 투자자에게 제공한다.

### 핵심 가치
- **선행성**: 가격 변동 전에 매수/매도 압력의 방향 감지 — 개인 투자자에게 "0.5초의 우위"
- **분석 깊이**: 단순 잔량이 아닌 잔량의 변화율, 스푸핑(허수 주문) 의심 패턴까지 분석
- **실시간성**: 장중 분봉 단위로 호가 압력 변동을 추적하여 진입/이탈 타이밍 최적화

### 작동 방식 (핵심 분석)

#### 1단계: 호가 압력 스코어 계산
```
호가 압력 스코어 (Order Pressure Score, OPS):

공식: OPS = (매수 잔량 상위 5호가 합) / (매도 잔량 상위 5호가 합)

해석:
• OPS > 1.5 → 강한 매수 압력 (매수벽)
• OPS 0.8~1.5 → 중립
• OPS < 0.8 → 강한 매도 압력 (매도벽)

예시: SK하이닉스 (000660) — 현재 OPS 추이
09:15 OPS 1.1 (중립)
09:20 OPS 1.4 (매수 압력 증가 중)
09:25 OPS 1.8 (매수벽 형성) ← 알림 발생
09:28 가격 +1.2% 상승 시작 ← 3분 선행

핵심: OPS가 1.5를 돌파하는 "순간"이 진입 시그널
```

#### 2단계: 스푸핑(허수 주문) 의심 패턴 감지
```
스푸핑 패턴 판정 기준:
① 특정 호가에 대량 매수(매도) 주문 등장
② 가격이 해당 방향으로 이동하면 해당 주문 취소
③ 1~2분 내 반복 발생 (3회 이상)

예시:
09:30 매수 1호가에 50,000주 출현 (OPS 2.1로 급등)
09:31 가격 +0.3% 상승
09:31 매수 1호가 50,000주 전량 취소 → OPS 1.0으로 급락
09:32 매도 1호가에 40,000주 출현
→ "스푸핑 의심 — 허수 매수벽으로 개인 매수 유도 후 매도"

경고: 🚨 OPS 급변 종목 — 허수 주문 가능성 주의
```

#### 3단계: 호가 압력 + 가격 괴리 분석
```
호가 압력 vs 가격 방향 괴리:

┌────────────┬──────────┬──────────┬────────────────┐
│ 상황        │ OPS 방향  │ 가격 방향 │ 해석           │
├────────────┼──────────┼──────────┼────────────────┤
│ 강세 확인   │ 매수 압력↑│ 가격 상승 │ 추세 지속 가능  │
│ 숨은 매집   │ 매수 압력↑│ 가격 횡보 │ 곧 상승 가능성  │
│ 고점 경고   │ 매도 압력↑│ 가격 상승 │ 상승 한계 근접  │
│ 바닥 징후   │ 매수 압력↑│ 가격 하락 │ 반등 임박 가능  │
└────────────┴──────────┴──────────┴────────────────┘

가장 유용한 패턴: "숨은 매집" — 가격은 안 움직이는데 매수벽이 쌓임
→ 기관의 조용한 매집 신호일 가능성 높음
```

### 텔레그램 알림 예시
```
📊 [호가 압력] 매수벽 형성 감지
━━━━━━━━━━━━━━━━━

📌 SK하이닉스 (000660)
• OPS: 1.1 → 1.8 (10분간 급상승)
• 매수 잔량: 320,000주 (평소 150,000)
• 가격: 192,500원 (횡보 중)
• 패턴: "숨은 매집" — 가격 정체 + 매수벽 형성

📊 과거 유사 패턴 (OPS 1.8+ & 가격 횡보):
• 18건 중 13건 → 30분 내 +1% 이상 상승 (72%)
• 평균 상승폭: +1.8% (30분 기준)

⚠️ 스푸핑 위험: 낮음 (잔량 유지 시간 8분+)

💡 판단: 기관 매집 패턴 가능성.
   현재가 근처 지지 확인 시 단기 진입 검토.
   OPS 1.5 이탈 시 매집 실패로 판단, 이탈.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| KIS API | signal_analysis → 실시간 호가 잔량 데이터 조회 |
| 분봉 데이터 | theme_analysis → 호가 압력과 가격 변동 시계열 매칭 |
| 거래량 프로파일 | theme_analysis → 정상 거래량 대비 호가 잔량 이상 여부 |
| 투자자별 매매동향 | theme_analysis → OPS 이상과 기관/외국인 매매 교차 분석 |
| 매매 신호 | signal_analysis → 호가 압력 + 기술적 신호 결합 |
| 텔레그램 봇 | 장중 OPS 이상 감지 시 즉시 발송 |

### 기술 구현
- **호가 수집**: KIS API 실시간 호가 데이터 (5초~30초 간격 폴링)
- **OPS 계산**: 매수/매도 5호가 잔량 합계 비율 산출, 이동평균으로 노이즈 제거
- **스푸핑 감지**: 대량 주문 등장 → 취소 패턴을 시계열로 추적
- **괴리 분석**: OPS 방향 vs 가격 방향 비교, 괴리 발생 시 알림
- **알림**: OPS 1.5 돌파 또는 스푸핑 의심 시 텔레그램 전송
- **GitHub Actions**: 장중 실행 (cron-job.org 5분 간격 트리거)

### 난이도 & 기대효과
- **난이도**: ★★★★☆ (KIS API 호가 데이터 실시간 폴링 + 시계열 분석 로직)
- **기대효과**: 개인 투자자가 접근하기 어려운 마이크로스트럭처 분석을 자동화. "가격이 움직이기 전" 방향성 감지로 진입 타이밍 최적화. 스푸핑 경고로 허수 주문에 속지 않는 방어 능력 확보.

## 아이디어 27: 대주주/내부자 거래 추적기 (Insider Activity Tracker)

### 개요
상장사 대주주, 임원, 주요주주의 **자사주 매수/매도 공시**를 실시간으로 수집하여 "내부자가 자기 회사 주식을 사고 있는가?"를 추적하는 시스템. 내부자는 자사의 실적과 전망을 가장 잘 아는 사람이므로, 내부자의 매수는 가장 강력한 "정보 우위 신호" 중 하나다.

### 핵심 가치
- **정보 비대칭 활용**: 내부자의 행동은 시장에 아직 반영되지 않은 긍정적/부정적 전망의 "행동 증거"
- **분석 깊이**: 단순 공시 수집이 아닌 매수 규모/빈도/타이밍의 이상 패턴 감지
- **실시간성**: DART 공시 즉시 파싱 → 텔레그램 발송 (공시 후 10분 이내)

### 작동 방식 (핵심 분석)

#### 1단계: 내부자 거래 공시 수집
```
DART 공시 모니터링 대상:
• "임원·주요주주특정증권등소유상황보고서"
• "주식등의대량보유상황보고서" (5% 룰)
• "자기주식취득/처분결정" (자사주 매입)

파싱 항목:
┌──────────────┬────────────────────────┐
│ 항목          │ 추출 데이터             │
├──────────────┼────────────────────────┤
│ 종목명/코드   │ 보고 대상 회사          │
│ 보고자        │ CEO/CFO/사외이사/5%주주 │
│ 거래 유형     │ 매수/매도/증여/상속      │
│ 거래 수량     │ 주식 수                 │
│ 거래 금액     │ 거래 총액               │
│ 보유 비율 변동│ 변경 전 → 변경 후        │
│ 거래 사유     │ 경영 참여/단순투자 등    │
└──────────────┴────────────────────────┘
```

#### 2단계: 내부자 매수 시그널 스코어링
```
내부자 신호 강도 스코어 (0~100):

가중 항목:
1. 보고자 직급 (30%)
   • CEO/대표이사 매수 = 30점
   • CFO/재무 임원 매수 = 25점
   • 사외이사 매수 = 15점
   • 5% 주요주주 매수 = 20점

2. 거래 규모 (30%)
   • 연봉 대비 50%+ = 30점 (확신의 베팅)
   • 연봉 대비 20~50% = 20점
   • 연봉 대비 20% 미만 = 10점

3. 거래 패턴 (20%)
   • 3개월 내 2회+ 연속 매수 = 20점 (반복 확신)
   • 단발 매수 = 10점
   • 실적 발표 D-30 이내 = 15점 (타이밍 의미)

4. 시장 맥락 (20%)
   • 주가 하락 중 매수 = 20점 (역발상 = 강한 확신)
   • 주가 횡보 중 매수 = 15점
   • 주가 상승 중 매수 = 5점 (의무 매수 가능성)

예시: 삼성바이오로직스 CEO 자사주 매수
• 직급: CEO = 30점
• 규모: 15억원 (연봉 대비 약 60%) = 30점
• 패턴: 2개월 전에도 10억 매수 (연속) = 20점
• 맥락: 주가 3개월 -18% 하락 중 = 20점
→ 내부자 신호 스코어: 100/100 (극강)
```

#### 3단계: 내부자 매수 후 수익률 통계
```
과거 6개월 내부자 매수 후 성과 (스코어별):

스코어 80+ (강한 확신):
• 표본: 15건
• D+30 평균: +8.2% (승률 80%)
• D+60 평균: +12.5% (승률 73%)

스코어 50~79 (보통):
• 표본: 42건
• D+30 평균: +3.1% (승률 62%)
• D+60 평균: +4.8% (승률 57%)

스코어 50 미만 (약함/형식적):
• 표본: 68건
• D+30 평균: +0.5% (승률 51%)
→ 시장 평균과 유의미한 차이 없음
```

### 텔레그램 알림 예시
```

📈 과거 유사 패턴 (스코어 80+):
• 15건 중 D+30 평균 +8.2% (승률 80%)
• 최대: +22.5% (셀트리온 CEO 매수, 2025-09)

💡 판단: CEO가 3개월 하락 중에도 연속 대량 매수.
   내부 실적 전망에 대한 강한 확신 시그널.
   단, DART 공시 기준 — 시장 반영까지 1~3일 소요.
━━━━━━━━━━━━━━━━━
```

- **난이도**: ★★★☆☆ (DART API 무료 제공, 공시 파싱은 정형화되어 있음)
- **기대효과**: "내부자가 사고 있다"는 시장에서 가장 신뢰할 수 있는 비가격 신호. 기관 리서치에서 활용하는 인사이더 트래킹을 개인 투자자에게 자동 제공. 중장기 투자 종목 선정에 "경영진 확신"이라는 강력한 레이어 추가.

## 아이디어 28: 기관 컨센서스 괴리 탐지기 (Consensus Drift Detector)

### 개요
증권사 애널리스트들의 **목표가/투자의견 변화 추이**를 추적하여, 컨센서스가 "조용히 상향되고 있는" 종목과 "슬그머니 하향되고 있는" 종목을 선제적으로 감지하는 시스템. 목표가 변경은 보통 개별 리포트 단위로 보도되지만, **다수 증권사의 방향이 일치하는 순간**이 진짜 시그널이다.

### 핵심 가치
- **정보 집약**: 흩어져 있는 개별 증권사 리포트를 하나의 "컨센서스 방향" 지표로 집약
- **선행성**: 목표가 상향이 누적되면 주가가 따라가는 경향 — 컨센서스 드리프트가 주가를 선행
- **편의성**: "어떤 종목에 대해 애널리스트들이 조용히 의견을 바꾸고 있는가?" 자동 탐지

### 작동 방식 (핵심 분석)

#### 컨센서스 드리프트 계산
```
컨센서스 드리프트 스코어 (CDS):

1. 최근 30일 내 목표가 변경 리포트 수집
2. 각 리포트의 변경 방향 분류:
   상향 (+1) / 유지 (0) / 하향 (-1)
3. 가중 방향 점수:
   CDS = Σ(방향 × 증권사 영향력) / 전체 리포트 수

해석:
• CDS > +0.6 → 강한 상향 드리프트 (다수 증권사 상향)
• CDS -0.3~+0.3 → 혼조 또는 변화 없음
• CDS < -0.6 → 강한 하향 드리프트 (다수 증권사 하향)

예시: SK하이닉스 최근 30일 목표가 변경
┌──────────────┬──────────┬──────────┬────────┐
│ 증권사        │ 이전 목표가│ 변경 목표가│ 방향   │
├──────────────┼──────────┼──────────┼────────┤
│ 삼성증권      │ 210,000  │ 235,000  │ 상향 ↑ │
│ 미래에셋      │ 220,000  │ 240,000  │ 상향 ↑ │
│ 한국투자      │ 200,000  │ 225,000  │ 상향 ↑ │
│ KB증권        │ 215,000  │ 215,000  │ 유지 — │
│ NH투자        │ 195,000  │ 220,000  │ 상향 ↑ │
└──────────────┴──────────┴──────────┴────────┘

CDS = (4 × 상향 + 0 × 유지) / 5 = +0.80 (강한 상향 드리프트)
현재 주가: 192,500원 | 평균 목표가: 227,000원 (괴리율 +18%)
```

#### 주가 vs 컨센서스 괴리 분석
```
📊 컨센서스 괴리 매트릭스

              컨센서스 방향 →
           하향        유지        상향
      ┌──────────┬──────────┬──────────┐
높음  │ 🔴 급락  │ 과대평가 │ 혼조     │
      │ 위험    │ 주의     │         │
가 ├──────────┼──────────┼──────────┤
격 적정│ 하방 압력│ 안정     │ ★ 상승  │
괴      │         │          │ 임박   │
리 ├──────────┼──────────┼──────────┤
↑ 낮음│ 약세    │ 저평가?  │ ★★ 최적│
      │ 전환    │ 관찰     │ 진입점  │
      └──────────┴──────────┴──────────┘

★★ 최적 진입: 목표가 대비 괴리율 높은데 + 컨센서스 상향 중
→ "애널리스트들은 올린다고 하는데 주가가 아직 안 올랐다"
```

### 텔레그램 알림 예시
```
📈 [컨센서스] 상향 드리프트 감지
━━━━━━━━━━━━━━━━━

📌 SK하이닉스 (000660)
• 컨센서스 드리프트 스코어: +0.80 (강한 상향)
• 30일간: 상향 4건 | 유지 1건 | 하향 0건
• 평균 목표가: 227,000원 (30일 전 209,000 → +8.6%)
• 현재 주가: 192,500원 (목표가 대비 -15.2% 할인)

📊 과거 유사 패턴 (CDS 0.7+ & 괴리 15%+):
• 21건 중 16건 → D+30 목표가 방향 수렴 (76%)
• 평균 D+30 수익: +6.8%

🔑 최근 상향 사유 (공통):
"HBM4 수주 확대 + AI 서버 수요 예상 상회"

💡 판단: 5개 증권사 중 4곳이 목표가 상향.
   주가는 아직 미반영. 컨센서스 수렴 구간 진입 예상.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| Naver News API | 증권사 리포트/목표가 변경 뉴스 수집 |

| 투자자별 매매동향 | theme_analysis → 목표가 상향과 기관 순매수 동조 확인 |
| Gemini AI | 목표가 변경 사유 요약 + 자연어 해석 |
| Supabase | 증권사별 목표가 이력 DB 누적 |

### 기술 구현
- **데이터 수집**: Naver 증권 컨센서스 페이지 크롤링 또는 증권사 리포트 RSS
- **파서**: 증권사명/목표가/투자의견/날짜 구조화 추출
- **CDS 계산**: 30일 윈도우 방향성 가중합, 일별 업데이트
- **괴리율**: 평균 목표가 vs 현재 주가 비율
- **알림**: CDS ±0.6 이상 변동 감지 시 텔레그램 전송
- **GitHub Actions**: 일 1회 장 마감 후 컨센서스 데이터 갱신

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (Naver 증권 컨센서스 데이터 접근 용이, 파싱 + 통계 구축)
- **기대효과**: 개별 리포트를 일일이 찾아볼 필요 없이 "시장 전문가 집단의 방향" 한눈에 파악. 컨센서스 상향 + 주가 미반영 = 개인 투자자의 정보 시차 활용 기회. 기관과 같은 눈높이의 종목 선정 가능.

## 아이디어 29: 테마 전이 예측기 (Theme Contagion Predictor)

### 개요
한 테마의 대장주가 급등할 때, **다음으로 움직일 관련 종목(후발주)을 예측**하는 시스템. 테마 투자의 핵심은 "대장주 다음에 뭐가 오르나?"인데, 과거 패턴을 분석하면 **테마 내 전이 순서(contagion order)가 반복되는 경향**이 있다. 이 순서를 데이터로 학습하여 후발주 진입 타이밍을 잡는다.

### 핵심 가치
- **예측력**: "대장주가 오르기 시작했으니 다음은 이 종목" — 1~2시간 선행 예측
- **분석 깊이**: 테마 내 종목 간 "시차 상관관계"를 과거 90일 데이터로 학습
- **편의성**: 테마 급등 시 "지금 사야 할 후발주 3선" 자동 추천

### 작동 방식 (핵심 분석)

#### 1단계: 테마 전이 순서 학습
```
과거 90일간 "AI반도체" 테마 급등 사례 (5회):

급등 순서 분석 (대장주 급등 후 후발주 반응 시차):
┌──────────────┬────────┬────────┬────────┬────────┐
│ 종목          │ 1차 급등│ 2차 급등│ 3차 급등│ 시차   │
├──────────────┼────────┼────────┼────────┼────────┤
│ SK하이닉스   │ D+0    │ D+0    │ D+0    │ 대장주 │
│ 한미반도체   │ D+0    │ D+1    │ D+0    │ 평균 0.2일│
│ 리노공업     │ D+1    │ D+1    │ D+1    │ 평균 1.0일│
│ ISC          │ D+1    │ D+2    │ D+1    │ 평균 1.3일│
│ 하나마이크론 │ D+2    │ D+2    │ D+2    │ 평균 2.0일│
│ 네패스       │ D+2    │ D+3    │ D+2    │ 평균 2.3일│
└──────────────┴────────┴────────┴────────┴────────┘

핵심 발견:
• 대장주(SK하이닉스) 급등 후 후발주 전이 순서가 반복됨
• 한미반도체는 거의 동시(D+0~1), 네패스는 D+2~3에 반응
• 전이 시차의 표준편차가 작을수록 예측 신뢰도 높음
```

#### 2단계: 실시간 전이 감지 & 후발주 추천
```
📊 테마 전이 감지 (2026-03-15 10:30)
━━━━━━━━━━━━━━━━━

🔥 트리거: SK하이닉스 +4.5% 급등 (AI반도체 테마)

후발주 전이 예측:
┌──────────────┬──────────┬──────────┬──────────┐
│ 종목          │ 예상 시차 │ 현재 변동 │ 전이 확률│
├──────────────┼──────────┼──────────┼──────────┤
│ 한미반도체   │ D+0 (오늘)│ +1.2%   │ 92%      │
│ 리노공업     │ D+1 (내일)│ +0.3%   │ 78%      │
│ ISC          │ D+1 (내일)│ -0.1%   │ 72%      │
│ 하나마이크론 │ D+2      │ +0.1%   │ 65%      │
└──────────────┴──────────┴──────────┴──────────┘

→ 한미반도체: 이미 +1.2% 반응 시작, 과거 5회 중 5회 동일일 전이
→ 리노공업: 아직 미반응, 내일 전이 가능성 78%
   → ★ 오늘 장 마감 전 진입 타이밍
```

#### 3단계: 전이 강도 분석
```
전이 강도 (대장주 상승률 대비 후발주 추종률):

대장주 +3% 급등 시 후발주 평균 반응:
• 1순위 후발주: 대장주의 70~90% 추종 (한미반도체 +2.1~2.7%)
• 2순위 후발주: 대장주의 50~70% 추종 (리노공업 +1.5~2.1%)
• 3순위 후발주: 대장주의 30~50% 추종 (ISC +0.9~1.5%)

⚠️ 전이 실패 조건 (과거 추종 안 된 경우):
• 대장주만 특수 재료(수주 공시 등) → 테마 전이 약함
• 전체 시장 약세 (심리 온도 30 이하) → 전이 차단
• 후발주 자체 악재 (공매도 급증, 실적 하향)
```

### 텔레그램 알림 예시
```
🔗 [테마 전이] AI반도체 후발주 매수 타이밍
━━━━━━━━━━━━━━━━━

🔥 대장주 급등: SK하이닉스 +4.5% (10:30)

📋 후발주 진입 추천 (전이 확률순):
1. 한미반도체 — 전이 확률 92% (D+0)
   현재 +1.2% | 과거 추종률 80% | 예상 +3.6%
   → 이미 전이 시작. 눌림목 대기 또는 즉시 진입.

2. 리노공업 — 전이 확률 78% (D+1)
   현재 +0.3% | 과거 추종률 60% | 예상 +2.7%
   → 오늘 장 마감 전 선제 진입 검토.

3. ISC — 전이 확률 72% (D+1)
   현재 -0.1% | 과거 추종률 55% | 예상 +2.5%
   → 내일 장 초반 갭 상승 여부로 판단.

⚠️ 전이 실패 주의 사항:
• SK하이닉스 급등 원인: SOX +2.1% (테마 전체 호재)
  → 개별 재료 아님. 전이 가능성 높음 ✓
• 시장 심리 온도: 62 (중립~긍정) → 전이 차단 없음 ✓

💡 전략: "대장주 동시 진입은 늦음. 리노공업 D+1 전이
   패턴 활용하여 오늘 장 마감 전 선제 포지션 구축."
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 테마 분류 + TOP30 | theme_analysis → 테마별 종목 리스트 + 대장주 식별 |
| 종목 시세 히스토리 (90일) | theme_analysis → 테마 내 종목 간 시차 상관관계 학습 |
| 분봉 데이터 | theme_analysis → 장중 전이 시작 감지 (분 단위 정밀도) |
| 거래량 프로파일 | theme_analysis → 전이 시작의 거래량 급증 확인 |
| 투자자별 매매동향 | theme_analysis → 후발주로 수급 이동 패턴 확인 |
| Gemini AI 테마 분석 | theme_analysis → 전이 원인/맥락 자연어 해석 |

### 기술 구현
- **전이 순서 학습**: 90일 히스토리에서 테마별 대장주 급등일 추출 → 후발주 반응 시차 계산
- **시차 상관관계**: 종목 A의 D+0 수익률 vs 종목 B의 D+N 수익률 상관계수 (N=0~5)
- **실시간 트리거**: 대장주 급등(+3% 이상) 감지 → 전이 예측 모델 실행
- **전이 확률**: 과거 전이 발생 빈도 / 대장주 급등 횟수
- **알림**: 대장주 급등 감지 시 후발주 추천 텔레그램 전송
- **GitHub Actions**: 장중 cron-job.org 10분 간격 + 장 마감 후 전이 모델 재학습

### 난이도 & 기대효과
- **난이도**: ★★★★☆ (시차 상관관계 분석 + 실시간 트리거 로직)
- **기대효과**: 테마 투자의 가장 큰 고민인 "대장주는 이미 올랐는데 뭘 사지?"에 데이터 기반 답변. 후발주 1~2일 선행 진입으로 수익 극대화. "다음 차 종목"을 감이 아닌 통계로 선정.

## 아이디어 30: 시간대별 수익률 히트맵 (Intraday Alpha Map)

### 개요
종목별/섹터별로 **특정 시간대에 반복적으로 수익이 발생하는 패턴**을 히트맵으로 시각화하는 시스템. "이 종목은 14시에 항상 강하다", "반도체는 오전 장 초반에 약하고 오후에 강하다"와 같은 **인트라데이 알파**를 발굴한다. 대부분의 분석이 "일봉" 단위인데, 분봉 레벨의 시간대 패턴은 거의 분석되지 않는 숨겨진 영역이다.

### 핵심 가치
- **발굴 난이도 높은 알파**: 일봉 분석으로는 절대 보이지 않는 장중 시간대 패턴
- **실시간성**: "지금 이 시간, 이 종목은 통계적으로 강한(약한) 시간대" 안내
- **편의성**: 히트맵 한 장으로 종목의 장중 리듬을 직관적으로 파악

### 작동 방식 (핵심 분석)

#### 1단계: 시간대별 수익률 집계
```
종목별 30분 단위 평균 수익률 (최근 60일 기준):

SK하이닉스 시간대별 수익률 히트맵:
┌────────┬────────┬──────────┬────────┐
│ 시간대  │ 평균등락│ 승률     │ 강도   │
├────────┼────────┼──────────┼────────┤
```

## 아이디어 31: 동시호가 이상 감지기 (Auction Anomaly Detector)

### 개요
장 시작 동시호가(08:30~09:00)와 장 마감 동시호가(15:20~15:30)에서 발생하는 **비정상적 주문 패턴**을 감지하는 시스템. 동시호가는 "프리뷰 시장"으로, 여기서 대량 주문이 들어오면 정규장 방향성의 강력한 단서가 된다. 특히 마감 동시호가의 이상은 다음 날 장을 예고한다.

### 핵심 가치
- **초기 선행성**: 정규장 시작 전 30분간의 주문 흐름으로 개장 방향 예측
- **분석 깊이**: 동시호가 예상 체결가 변동 + 잔량 비대칭 + 기관 바스켓 주문 감지
- **편의성**: "이 종목은 동시호가에서 이상 매수세가 들어오고 있다" 자동 탐지

### 작동 방식 (핵심 분석)

#### 1단계: 시초가 동시호가 분석 (08:30~09:00)
```
동시호가 이상 감지 기준:

1. 예상 체결가 급변동
   • 08:40 예상가 195,000 → 08:55 예상가 199,500 (+2.3%)
   • 정상: 예상가 변동 ±0.5% 이내
   → 15분간 +2.3% 급변 = 대형 매수 유입 의심

2. 매수/매도 잔량 비대칭
   • 매수 호가 잔량: 2,500,000주
   • 매도 호가 잔량:   800,000주
   • 비율: 3.1:1 (정상 1.2~1.8:1)
   → 극단적 매수 우위 = 갭 상승 확률 극대

3. 기관 바스켓 주문 추정
   • 동일 테마 5종목 이상 동시에 예상가 +1%+ 상승
   → 기관 프로그램 매매의 테마 바스켓 주문 가능성

예시: AI반도체 테마 동시호가 (08:55)
┌──────────────┬──────────┬──────────┬──────────┐
│ 종목          │ 예상 변동 │ 잔량 비율 │ 이상 여부│
├──────────────┼──────────┼──────────┼──────────┤
│ SK하이닉스   │ +2.3%    │ 3.1:1    │ 🔴 이상  │
│ 한미반도체   │ +1.8%    │ 2.4:1    │ 🟡 주의  │
│ 삼성전자     │ +0.8%    │ 1.6:1    │ ⬜ 정상  │
│ 리노공업     │ +1.5%    │ 2.1:1    │ 🟡 주의  │
└──────────────┴──────────┴──────────┴──────────┘
→ AI반도체 3종목 동시 이상 = 기관 바스켓 가능성
```

#### 2단계: 마감 동시호가 분석 (15:20~15:30)
```
마감 동시호가 이상 → 다음날 예측:

패턴 A: 마감 대량 매수 (종가 끌어올리기)
• 15:20~15:30 사이 종가 예상가 +1.5% 급등
• 원인: 기관/외국인 종가 매수 (벤치마크 조정, 펀드 설정 등)
• 다음날: 추가 상승 확률 62% (관성)

패턴 B: 마감 대량 매도 (종가 밀어내리기)
• 15:20~15:30 사이 종가 예상가 -2.0% 급락
• 원인: 대량 청산, 손절, 리밸런싱
• 다음날: 반등 확률 55% (과매도 반응)

패턴 C: 마감 잔량 폭발 (평소 3배+)
• 체결 잔량이 평소의 3배 이상
• 원인: 지수 리밸런싱, ETF 설정/환매
• 다음날: 방향은 불확실하나 변동성 확대 예상
```

## 아이디어 32: 프로그램 매매 역추적기 (Program Trade Reverse Tracker)

### 개요
기관의 프로그램 매매(차익/비차익)의 **방향 전환 시점**을 포착하여, 기관이 "오늘 매수 프로그램을 돌리기 시작했다" 또는 "매도 프로그램으로 전환했다"를 실시간으로 감지하는 시스템. 프로그램 매매는 장중 지수 방향의 60~70%를 좌우하는 핵심 변수인데, 개인 투자자는 결과(종가)만 볼 수 있고 과정(장중 전환)은 놓친다.

### 핵심 가치
- **시장 미시구조 이해**: 장중 지수 급변의 원인을 실시간으로 추적 — "왜 갑자기 떨어지지?" 즉답
- **선행성**: 프로그램 매도 전환 시점 포착 → 보유 종목 방어적 대응
- **분석 깊이**: 차익/비차익 구분 + 누적 금액 + 전환 시점의 시장 조건 분석

### 작동 방식 (핵심 분석)

#### 프로그램 매매 흐름 실시간 추적
```
프로그램 매매 실시간 모니터 (장중):

시간   차익     비차익    누적      지수변동
09:30  +120억   +350억   +470억    +0.3%
10:00  +180억   +580억   +760억    +0.6%
10:30  +200억   +820억   +1,020억  +0.8%
11:00  +150억   +450억   +1,170억  +0.7%
13:00  -80억    +200억   +1,290억  +0.5% ← 차익 전환!
13:30  -250억   -100억   +940억    +0.2% ← 비차익도 전환!
14:00  -400억   -350억   +190억    -0.3% ← 프로그램 순매도

🚨 전환 감지: 13:00 차익 매도 전환, 13:30 비차익 동반 전환
→ 프로그램 순매수 +1,290억에서 30분 만에 +190억으로 급감
→ 지수도 +0.7% → -0.3%로 반전
```

#### 프로그램 전환 시그널 판정
```
프로그램 매매 전환 시그널:

레벨 1 (주의): 차익 매매 방향 전환
• 차익 순매수 → 순매도 전환 시점
• 의미: 선물-현물 차익 관계 변화, 초기 경고

레벨 2 (경고): 비차익 매매 동반 전환
• 차익 + 비차익 모두 매도 전환
• 의미: 기관 전체의 시장 방향 변경 판단
• 과거 통계: 동반 전환 후 30분 내 -0.5% 추가 하락 68%

레벨 3 (긴급): 프로그램 폭탄 매도
• 30분 내 프로그램 순매도 -500억 이상
• 의미: 기관 대규모 청산 또는 헤지
• 과거 통계: 당일 종가 하락 확률 82%

예시: 2026-03-14 레벨 2 시그널 발생 (13:30)
→ 30분 후 KOSPI -0.7% 추가 하락
→ 보유 종목 평균 -1.2% 손실
→ 시그널 수신 시 방어 조치 가능했던 구간
```

### 텔레그램 알림 예시
```
🚨 [프로그램] 매도 전환 감지 (레벨 2)
━━━━━━━━━━━━━━━━━

⚠️ 프로그램 매매 방향 전환:
```

## 아이디어 33: 수급 클러스터 분석기 (Flow Cluster Analyzer)

### 개요
외국인/기관/개인 3주체의 매매 데이터를 **시계열 클러스터링**하여, "지금 시장은 어떤 수급 국면인가?"를 자동 분류하는 시스템. 단순히 "외국인 매수/매도"를 보는 것이 아니라, 3주체의 **조합 패턴**이 과거 어떤 시장 국면과 유사한지를 매칭한다.

### 핵심 가치
- **패턴 인식**: "외국인 매수 + 기관 매도 + 개인 매도" = 과거에 이런 조합일 때 시장은 이랬다
- **분석 깊이**: 3주체 수급의 조합 패턴을 과거 데이터와 매칭, 단순 수급보다 깊은 해석
- **편의성**: "지금 수급 구조가 2024년 11월 랠리 시작 때와 유사합니다" 자동 판별

## 아이디어 34: 손절/익절 최적화 엔진 (Exit Strategy Optimizer)

### 개요
"언제 팔아야 하는가?"는 투자에서 가장 어려운 질문이다. 이 시스템은 종목별/전략별로 **과거 데이터 기반 최적 손절선과 익절선**을 자동으로 산출하고, 보유 중인 종목에 실시간으로 적용한다. "내가 설정한 -3% 손절이 이 종목에 맞는 전략인가?"에 데이터로 답한다.

### 핵심 가치
- **개인화**: 종목별 변동성에 맞춘 최적 손절/익절선 (일률적 -3% 아님)
- **분석 깊이**: 수백 건의 과거 매매를 시뮬레이션하여 최적 비율 도출
- **편의성**: 매수 시 자동으로 "이 종목의 최적 손절은 -X%, 익절은 +Y%"를 제안

### 작동 방식 (핵심 분석)

#### 1단계: 종목별 최적 손절/익절 비율 산출
```
최적 비율 산출 방법 (몬테카를로 시뮬레이션):

1. 종목의 90일 일봉 데이터에서 모든 매수 시점 생성
2. 각 매수 시점에서 다양한 손절/익절 비율 조합 적용
3. 조합별 수익률/승률/최대손실 계산
4. 기대수익 최대화 + 최대손실 최소화 조합 선정

SK하이닉스 최적 비율 (90일 시뮬레이션):
┌────────┬────────┬────────┬────────┬──────────┐
│ 손절    │ 익절   │ 승률   │ 기대수익│ 최대손실 │
├────────┼────────┼────────┼────────┼──────────┤
│ -2%    │ +3%    │ 58%    │ +0.5%  │ -2.0%    │
│ -3%    │ +5%    │ 52%    │ +1.2%  │ -3.0%    │
│ -4%    │ +7%    │ 48%    │ +1.8%  │ -4.0%    │ ← ★ 최적
│ -5%    │ +10%   │ 42%    │ +1.5%  │ -5.0%    │
│ -7%    │ +15%   │ 35%    │ +1.1%  │ -7.0%    │
└────────┴────────┴────────┴────────┴──────────┘

★ SK하이닉스 최적: 손절 -4% / 익절 +7%
  (기대수익 +1.8% 최대, 승률 48%이나 익절>손절 비율로 보상)

비교: 에코프로 (고변동성 종목)
★ 에코프로 최적: 손절 -6% / 익절 +12%
  (변동성이 크므로 넓은 밴드가 최적)

→ 일률적 -3% 손절은 SK하이닉스에는 너무 타이트,
  에코프로에는 아직 여유가 있음
```

#### 2단계: 실시간 이탈 모니터링
```
보유 종목 이탈 모니터 (장중):

┌──────────────┬────────┬────────┬────────┬──────────┐
│ 종목          │ 매수가  │ 현재가  │ 손익률  │ 이탈 거리│
├──────────────┼────────┼────────┼────────┼──────────┤
│ SK하이닉스   │ 190,000│195,500 │ +2.9%  │ 익절까지 │
│              │        │        │        │ +4.1%p  │
│ 에코프로     │ 85,000 │ 81,200 │ -4.5%  │ 손절까지 │
│              │        │        │        │ -1.5%p  │
│ 한화에어로   │ 320,000│335,000 │ +4.7%  │ 익절 임박│
│              │        │        │        │ ★ +0.3%p│
└──────────────┴────────┴────────┴────────┴──────────┘

⚠️ 한화에어로: 익절선 +5.0%까지 0.3%p — 트레일링 스탑 전환 권고
⚠️ 에코프로: 손절선 -6.0%까지 1.5%p — 반등 미발생 시 내일 손절 검토
```

#### 3단계: 트레일링 스탑 자동 조정
```
트레일링 스탑 로직:

종목이 익절선에 도달 후 추가 상승 시:
→ 고정 익절이 아닌 트레일링 스탑으로 자동 전환

예시: 한화에어로 (익절선 +5.0% 도달)
• 최고가 갱신: 336,000원 (+5.0%)
• 트레일링 스탑: 최고가 -2% = 329,280원
• 추가 상승: 345,000원 (+7.8%)
• 트레일링 갱신: 345,000 × 0.98 = 338,100원
→ 338,100원 이탈 시 매도 알림

트레일링 비율도 종목 변동성에 따라 자동 조정:
• 저변동 종목 (삼성전자): 트레일링 -1.5%
• 중변동 종목 (SK하이닉스): 트레일링 -2.0%
• 고변동 종목 (에코프로): 트레일링 -3.0%
```

### 텔레그램 알림 예시
```

━━━━━━━━━━━━━━━━━

📊 보유 종목 손절/익절 모니터:

✅ SK하이닉스: +2.9% (익절 +7.0%까지 4.1%p)
   최적 비율: 손절 -4% | 익절 +7% | 보유 유지

   → 현재가 335,000 | 트레일링 시작가 336,000
```

## 아이디어 35: 이벤트 캘린더 복합 분석기 (Event Calendar Compound Analyzer)

### 개요
FOMC, 고용지표, CPI, 옵션만기일, 쿼드러플위칭데이, 배당락일 등 **시장에 영향을 미치는 모든 이벤트를 캘린더에 통합**하고, 각 이벤트의 과거 시장 영향도를 통계로 제공하는 시스템. 핵심은 단일 이벤트가 아니라 **복수 이벤트가 겹치는 날의 복합 효과 분석**이다.

### 핵심 가치
- **선제적 대비**: "다음 주 목요일 = FOMC + 옵션만기 → 변동성 폭발 예상"
- **분석 깊이**: 이벤트 단독 효과 + 복합 효과 (2~3개 이벤트 동시 발생 시 시너지/상쇄)
- **편의성**: 주간/월간 이벤트 캘린더에 "이 날의 시장 영향도 스코어" 자동 표시

### 작동 방식 (핵심 분석)

### 로드맵 설계 원칙

1. **Phase 1**: 즉시 구현 가능 + 매일 사용 + 다른 기능의 기반 (수급 클러스터/이벤트 캘린더 추가)
2. **Phase 2**: 장중 실시간 가치 + 기존 인프라 위에 로직 추가 (프로그램 역추적/내부자 추적 추가)
3. **Phase 3**: 분석 깊이 강화 + 매도 전략 체계화 (손절익절 최적화/컨센서스/히트맵 추가)
4. **Phase 4**: 장기 데이터 축적 또는 높은 기술 난이도 (테마 전이/동시호가/호가 압력 추가)

### 5차 신규 아이디어 핵심 기여

| 영역 | 기존 1~4차 | 5차 추가 | 시너지 |
|------|-----------|---------|--------|
| 시장 구조 이해 | 심리 온도계, 섹터 흐름 | 수급 클러스터, 프로그램 역추적 | "왜 시장이 이렇게 움직이나"의 완전한 답변 |
| 매매 타이밍 | 크로스 시그널, 갭 분석 | 호가 압력, 동시호가, 시간대 히트맵 | 초(秒)/분 단위 진입 정밀도 향상 |
| 종목 선정 | 스마트 머니, 밸류 스크리너 | 내부자 추적, 컨센서스 괴리, 테마 전이 | 중장기 + 단기 선정 기준 다각화 |
| 매도 전략 | 리스크 모니터, 거래량 괴리 | 손절/익절 최적화 | "언제 파느냐"의 데이터 기반 답변 완성 |
| 이벤트 대비 | 프리마켓 모니터 | 이벤트 캘린더 복합 분석 | 주간/월간 단위 선제적 방어 체계 |
| 마이크로스트럭처 | 없음 | 호가 압력, 동시호가, 프로그램 역추적 | 기관만 보던 시장 미시구조를 개인에게 제공 |
