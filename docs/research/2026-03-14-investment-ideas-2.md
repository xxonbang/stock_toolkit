## 아이디어 6: 대화형 텔레그램 투자 어시스턴트

### 개요
텔레그램에서 자연어로 질문하면, 두 프로젝트의 **모든 데이터를 검색하여 AI가 답변**하는 대화형 챗봇. 대시보드를 열지 않고도 궁금한 정보를 즉시 확인할 수 있는 개인 투자 비서.

### 핵심 가치
- **편의성**: "삼성SDI 지금 어때?" 한마디로 종합 분석 수신
- **실시간성**: 장중 언제든 즉시 응답
- **분석 깊이**: 단순 데이터 조회가 아닌 AI 해석 포함

### 대화 예시
```
👤 "오늘 2차전지 테마 어때?"
🤖 "2차전지 테마는 현재 테마 순위 #2입니다.
    대장주 삼성SDI는 적극매수 신호(비전 85점), 외국인 3일 연속
    순매수 중입니다. 에코프로비엠은 공매도 비율 6.2%로 주의가
    필요합니다. 전체적으로 강세 유지 중이나, 어제 대비 모멘텀은
    소폭 둔화되었습니다."

👤 "외국인이 3일 이상 연속 매수 중인 종목 알려줘"
🤖 "현재 외국인 3일+ 연속 순매수 종목:
    1. SK하이닉스 - 5일 연속 (누적 +1,200억)
    2. 삼성SDI - 3일 연속 (누적 +450억)
    3. 현대차 - 4일 연속 (누적 +380억)
    이 중 매수 신호가 동시에 나온 종목은 SK하이닉스, 삼성SDI입니다."

👤 "지난주 적극매수 종목들 수익률은?"
🤖 "지난주(3/7~3/11) 적극매수 신호 종목 성과:
    • Vision 기준: 평균 +3.2% (5종목 중 4종목 상승)
    • Combined 기준: 평균 +4.1% (3종목 중 3종목 상승)
    최고: 한화에어로 +8.5% | 최저: 카카오 -1.2%"

👤 "/watchlist"
🤖 "현재 관심종목 5개:
    1. 삼성SDI - 적극매수 | 2차전지 #1
    2. SK하이닉스 - 매수 | AI반도체 #2
    ..."
```

### 지원 명령어

| 명령어 | 기능 |
|--------|------|
| 자연어 질문 | 종목/테마/시장에 대한 자유 질문 |
| `/stock 삼성SDI` | 종목 종합 카드 (신호+테마+수급+뉴스) |
| `/theme 2차전지` | 테마 상세 (순위+대장주+관련주 신호) |
| `/market` | 시장 현황 요약 (지수+F&G+VIX+환율) |
| `/top` | 오늘의 적극매수 종목 리스트 |
| `/watchlist` | 관심종목 목록 조회/관리 |
| `/history 삼성SDI` | 최근 30일 신호 변동 이력 |
| `/compare A B` | 두 종목 비교 분석 |

### 아키텍처
```
텔레그램 메시지
    ↓
Python 봇 서버 (webhook 또는 polling)
    ↓
의도 분류 (Gemini)
    ↓
┌─────────────────────────────┐
│ 데이터 검색 레이어            │
│ • theme_analysis JSON 파싱   │
│ • signal_analysis JSON 파싱  │
│ • history 디렉토리 검색       │
│ • 시뮬레이션 결과 조회        │
└─────────────────────────────┘
    ↓
Gemini로 자연어 응답 생성
    ↓
텔레그램 응답
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| `latest.json` (24.7MB) | 종목/테마 질문 응답의 주 데이터 |
| `combined_analysis.json` | 매매 신호 조회 |
| `theme-forecast.json` | 테마 예측 질문 응답 |
| history 디렉토리 (90일+) | 과거 데이터 질문 응답 |
| simulation 결과 | 수익률 질문 응답 |
| macro/fear_greed/vix JSON | 시장 현황 응답 |
| 텔레그램 봇 인프라 | 기존 봇 토큰 재활용 |
| Gemini API (5키) | 자연어 처리 + 응답 생성 |

### 기술 구현
- **봇 서버**: Python `python-telegram-bot` 라이브러리
- **호스팅**: GitHub Actions (polling 방식) 또는 무료 클라우드 (Railway/Render)
- **데이터 접근**: 두 프로젝트의 GitHub Pages에서 JSON fetch 또는 로컬 파일 읽기
- **캐싱**: 자주 묻는 질문에 대한 응답 캐싱 (API 비용 절감)
- **관심종목**: Supabase에 사용자별 watchlist 저장

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (텔레그램 봇 + Gemini 연동, 데이터 파싱)
- **기대효과**: 웹 대시보드를 열지 않고 이동 중에도 즉시 투자 정보 확인. 자연어 인터페이스로 데이터 접근 장벽 제거.

## 아이디어 7: 스마트 머니 수급 레이더

### 개요
외국인/기관 투자자의 매매 패턴을 **심층 추적**하여, 일반 개인 투자자보다 먼저 "스마트 머니"의 움직임을 포착하는 시스템. 단순 일별 순매수가 아닌, **연속성/규모/타이밍** 패턴을 분석.

### 핵심 가치
- **분석 깊이**: 수급 데이터를 단순 숫자가 아닌 패턴으로 해석
- **실시간성**: 수급 반전/이상 패턴 감지 시 즉시 알림
- **편의성**: 복잡한 수급 데이터를 직관적 시각화로 제공

### 핵심 1: 수급을 더 넘어 - 패턴 분석

기존 두 프로젝트는 수급 데이터를 "외국인 순매수 +300억" 같은 단일 수치로 보여줌. 이 프로젝트는 **패턴**을 분석:

| 패턴 유형 | 설명 | 의미 |
|-----------|------|------|
| **조용한 매집** | 소량씩 5일+ 연속 순매수, 주가 변동 적음 | 큰 상승 전 포지션 구축 가능성 |
| **급속 유입** | 1~2일 내 대규모 순매수 | 이벤트/재료 기반 단기 매매 가능성 |
| **쌍방 이탈** | 외국인+기관 동시 순매도 전환 | 강한 하락 신호 |
| **교차 수급** | 외국인 매수 + 기관 매도 (또는 반대) | 방향성 불확실, 관망 필요 |
| **거래량 동반 매집** | 순매수 + 거래량 증가 + 주가 상승 | 추세 확인 신호 |
| **프로그램 주도** | 프로그램 매매 비중 급등 | 기계적 매매, 단기 변동 주의 |

### 핵심 2: 스마트 머니 스코어

종목별로 수급 건전성을 0~100점으로 종합 평가:

```
스마트 머니 스코어 =
  외국인 연속 매수일수 (25%)
  + 기관 연속 매수일수 (20%)
  + 순매수 규모/시가총액 비율 (20%)
  + 매집 패턴 일치도 (20%)
  + 프로그램 매매 비중 역가중치 (15%)
```

### 텔레그램 알림 예시
```
🔍 [스마트 머니 포착] 조용한 매집 패턴
━━━━━━━━━━━━━━━━━

📌 한화에어로스페이스 (012450)
• 외국인: 8일 연속 순매수 (누적 +1,850억)
• 기관: 5일 연속 순매수 (누적 +420억)
• 주가 변동: 같은 기간 +2.1% (완만한 상승)
• 거래량: 평균 수준 유지 (급등 아님)
• 스마트 머니 스코어: 92/100

💡 해석: 대규모 자금이 주가 급등 없이 물량을
   조용히 확보 중. 과거 유사 패턴에서 이후 10거래일
   평균 +7.2% 상승.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 투자자별 매매동향 | theme_analysis → 외국인/기관/개인 일별 순매수 |
| 분봉 데이터 | theme_analysis → `intraday-history.json` (장중 수급 타이밍) |
| 거래량 프로파일 | theme_analysis → `volume-profile.json` |
| KIS 수급 데이터 | signal_analysis → 호가/매수매도 비율 |
| 프로그램 매매 데이터 | theme_analysis → stock_criteria (프로그램매매 지표) |
| 히스토리 (90일) | theme_analysis → history/ (수급 시계열 구축) |

### 기술 구현
- **데이터 파이프라인**: 히스토리 JSON에서 종목별 수급 시계열 추출
- **패턴 매칭**: 규칙 기반 패턴 탐지 (연속일수, 규모 임계값 등)
- **스코어링**: 가중 합산 → 상위 종목 필터링
- **백테스트**: 과거 패턴 발생 후 수익률 통계 (신뢰도 근거)
- **알림**: 신규 패턴 감지 시 텔레그램 전송
- **시각화**: 웹 페이지에 수급 히트맵 + 스코어 랭킹

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (히스토리 파싱 + 패턴 매칭 로직, 신규 데이터 수집 불필요)
- **기대효과**: 개인 투자자가 놓치기 쉬운 기관/외국인의 "조용한 움직임"을 체계적으로 포착. 단기/스윙 매매 모두에 활용 가능.

## 아이디어 8: 매매 일지 자동 생성 & AI 회고

### 개요
실제 매매 기록(KIS API 실계좌 또는 수동 입력)을 기반으로, AI가 **매매 일지를 자동 작성**하고 주기적으로 **투자 습관/편향을 분석**하여 피드백하는 시스템. "내가 잘한 것"과 "반복하는 실수"를 객관적으로 보여주는 개인 투자 코치.

### 핵심 가치
- **분석 깊이**: 매매 기록 + 당시 시장 상황을 결합한 깊은 회고
- **편의성**: 일지 작성 부담 제거 (자동 생성)
- **실시간성**: 매매 직후 즉시 기록 + 주간/월간 회고 자동 발송

### 작동 방식

#### 1단계: 매매 기록 수집
```
매매 기록 입력 방식:
A) KIS API 실계좌 연동 → 체결 내역 자동 수집
B) 텔레그램 수동 입력 → "/buy 삼성SDI 10주 425000"
C) CSV 파일 업로드 → 증권사 거래내역 파싱
```

#### 2단계: 맥락 자동 연결
매매 시점의 두 프로젝트 데이터를 자동 매칭:
```
매수 시점 맥락:
- 당시 매매 신호: 적극매수 (비전 85점)
- 소속 테마: 2차전지 (테마 순위 #1)
- 외국인 수급: 3일 연속 순매수
- F&G Index: 65 (탐욕)
- 매수 근거 추정: "테마 대장주 + 강한 매수 신호"
```

#### 3단계: AI 일지 자동 작성
```
📝 2026-03-14 (금) 매매 일지
━━━━━━━━━━━━━━━━━

📌 오늘의 매매
• 매수: 삼성SDI 10주 @ 425,000원
  - 매수 시점 신호: 적극매수 (비전 85, KIS 82)
  - 소속 테마: 2차전지 (#1)
  - 수급: 외국인 3일 연속 순매수
  ✅ 시스템 신호와 일치하는 매수

• 매도: 에코프로 5주 @ 98,000원 (수익률 +12%)
  - 매도 시점 신호: 중립 (비전 55, KIS 48)
  - 보유 기간: 8거래일
  ✅ 신호 악화 시점에 매도 — 적절한 타이밍

📊 당일 시장 상황
• KOSPI +1.2% | F&G 62 | VIX 18.5
• 2차전지 테마 강세 지속

💭 AI 코멘트
"오늘 매매는 시스템 신호에 부합합니다. 삼성SDI
매수는 테마 대장주 + 적극매수 신호 + 수급 호조가
모두 일치하는 이상적인 진입입니다."
━━━━━━━━━━━━━━━━━
```

#### 4단계: 주간/월간 AI 회고
```
📊 3월 2주차 주간 회고
━━━━━━━━━━━━━━━━━

📈 성과 요약
• 총 매매: 매수 4건, 매도 2건
• 실현 수익: +185,000원
• 평균 보유 기간: 5.2거래일
• 승률: 66.7% (3승 1패 2보유중)

🔍 패턴 분석
✅ 잘한 점:
  • 시스템 신호 일치율 85% (7/8 매매)
  • 손절 기준 준수 (-5% 도달 시 실행)

⚠️ 개선 포인트:
  • 추격 매수 경향: 3건 중 2건이 이미 +3% 이상
    상승한 시점에 매수. 눌림목 대기 전략 권장.
  • 테마 편중: 매수 4건 중 3건이 2차전지 관련.
    섹터 분산 필요.
  • 매도 타이밍: 에코프로 매도 시점은 적절했으나,
    카카오는 신호 악화 후 2일 지연 매도 → -3%
    추가 손실 발생.

📌 다음 주 체크포인트
• 추격 매수 자제 — 매수 시 "당일 등락률 < +2%" 규칙
• 테마 분산 — 동일 테마 최대 2종목 제한
• 매도 신호 즉시 실행 — 1거래일 내 대응
━━━━━━━━━━━━━━━━━
```

### 분석하는 투자 편향

| 편향 | 탐지 방법 | 피드백 |
|------|----------|--------|
| **추격 매수** | 매수가 > 당일 시가 +3% | "눌림목 대기 권장" |
| **손절 회피** | 매도 신호 후 3일+ 미실행 | "신호 대비 N일 지연, 추가 손실 X%" |
| **확증 편향** | 매수 신호만 따르고 매도 신호 무시 | "매도 신호 대응률 N%" |
| **과잉 매매** | 주 5회 이상 매매 | "매매 빈도 과다, 수수료 X원 발생" |
| **섹터 편중** | 동일 테마/섹터 3종목+ | "포트폴리오 N% 편중" |
| **손익비 불균형** | 평균 수익 < 평균 손실 | "이익은 빨리 실현, 손실은 방치하는 경향" |

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| 매매 신호 히스토리 (30일) | signal_analysis → history/ (매매 시점 신호 매칭) |
| 테마 분석 히스토리 (90일) | theme_analysis → history/ (매매 시점 테마 상황) |
| 시뮬레이션 수익률 | 두 프로젝트 → "신호를 따랐다면" 비교 벤치마크 |
| KIS API | 실계좌 체결 내역 조회 (기존 인증 인프라) |
| Gemini AI | 일지 작성 + 패턴 분석 + 피드백 생성 |
| Supabase | 매매 기록 + 회고 이력 저장 |

### 기술 구현
- **매매 기록**: KIS API `/uapi/domestic-stock/v1/trading/inquire-daily-ccld` (체결내역)
- **맥락 매칭**: 매매 날짜 기준으로 히스토리 JSON에서 당시 신호/테마 추출
- **AI 일지**: Gemini에 매매 기록 + 맥락 데이터 주입 → 구조화된 일지 생성
- **회고 분석**: 누적 매매 데이터에서 패턴 통계 계산 + AI 해석
- **알림**: 매매 발생 시 일지 자동 생성, 매주 일요일 주간 회고 전송
- **저장**: Supabase에 매매 기록 + 일지 + 회고 저장, 웹에서 열람 가능

### 난이도 & 기대효과
- **난이도**: ★★★★☆ (KIS 체결내역 API + 히스토리 매칭 + AI 분석)
- **기대효과**: 투자 실력 향상의 핵심인 "복기"를 자동화. 반복되는 실수 패턴을 객관적 데이터로 직면. 시스템 신호 대비 실제 매매의 차이를 정량적으로 측정.

## 아이디어 9: 뉴스 임팩트 분석기

### 개요
뉴스가 종목/테마 주가에 미치는 **실질적 영향도**를 정량 분석하는 시스템. "이 뉴스가 나온 후 주가가 실제로 얼마나 움직였는지"를 과거 데이터로 검증하고, 유사한 유형의 뉴스가 나왔을 때 **예상 임팩트**를 제시.

### 핵심 가치
- **분석 깊이**: 뉴스를 감성(긍정/부정)만이 아닌 실제 주가 영향도로 평가
- **실시간성**: 신규 뉴스 발생 시 과거 유사 뉴스의 임팩트 데이터 즉시 제공
- **편의성**: "이 뉴스에 반응해야 하나?"에 대한 데이터 기반 판단 근거

### 핵심 분석

#### 뉴스 유형 분류 & 임팩트 통계
```
┌──────────────────────────────────────────────┐
│ 📰 뉴스 임팩트 데이터베이스                       │
├──────────────┬──────────┬──────────┬─────────┤
│ 뉴스 유형     │ 평균 임팩트│ 지속 기간  │ 표본 수  │
├──────────────┼──────────┼──────────┼─────────┤
│ 실적 서프라이즈│ +4.2%    │ 3~5일    │ 45건    │
│ FDA 승인     │ +12.3%   │ 1~2일    │ 12건    │
│ 대규모 수주   │ +3.8%    │ 2~3일    │ 38건    │
│ 공매도 리포트  │ -8.5%    │ 1~3일    │ 15건    │
│ 정부 정책 발표 │ +2.1%    │ 5~10일   │ 60건    │
│ 증권사 목표가 │ +1.5%    │ 1일      │ 120건   │
│ 유상증자 공시  │ -5.2%    │ 1~2일    │ 25건    │
│ 경영권 분쟁   │ -3.1%    │ 변동     │ 8건     │
└──────────────┴──────────┴──────────┴─────────┘
```

#### 실시간 뉴스 분석 흐름
```
신규 뉴스 감지 (Naver News API)
    ↓
Gemini로 뉴스 유형 분류 + 관련 종목 추출
    ↓
과거 유사 뉴스 검색 (임팩트 DB)
    ↓
예상 임팩트 범위 산출
    ↓
해당 종목의 현재 매매 신호/테마 상태 결합
    ↓
텔레그램 알림
```

### 텔레그램 알림 예시
```
📰 [뉴스 임팩트] 삼성바이오 FDA 승인
━━━━━━━━━━━━━━━━━

📌 삼성바이오로직스 (207940)
뉴스: "삼성바이오 신약 FDA 승인 획득"
유형: FDA 승인

📊 과거 유사 뉴스 임팩트 (12건 분석)
• 평균 상승률: +12.3%
• 중앙값: +9.8%
• 범위: +5.2% ~ +22.1%
• 임팩트 지속: 평균 1.5거래일

🎯 현재 종목 상태
• 매매 신호: 매수 (비전 72점)
• 소속 테마: 바이오 (#3)
• 외국인: 2일 연속 순매수

💡 판단: 과거 데이터 기반 +10% 내외 상승 예상.
   단, 뉴스 임팩트는 1~2일 내 소멸하는 경향이므로
   단기 매매에 적합.
━━━━━━━━━━━━━━━━━
```

### 활용할 기존 자산
| 자산 | 용도 |
|------|------|
| Naver News API | 두 프로젝트에서 이미 연동 완료 → 뉴스 수집 |
| 뉴스 감성 분석 | signal_analysis → 종목별 뉴스 sentiment (-1.0~+1.0) |
| 종목 시세 히스토리 | theme_analysis → history/ (뉴스 전후 주가 변동 측정) |
| 분봉 데이터 | theme_analysis → intraday-history (뉴스 발행 시점 정밀 매칭) |
| Gemini AI | 뉴스 유형 분류 + 유사도 판단 |
| 매매 신호 | signal_analysis → 종목 현재 상태 결합 |

### 데이터 구축 과정
1. **과거 데이터 수집** (1회): 90일 히스토리에서 뉴스 + 주가 변동 쌍 추출
2. **유형 분류**: Gemini로 각 뉴스를 유형별 분류
3. **임팩트 계산**: 뉴스 발행일 기준 D+1, D+3, D+5 수익률 산출
4. **통계 구축**: 유형별 평균/중앙값/분포 계산
5. **지속 갱신**: 매일 새 데이터로 통계 업데이트

### 난이도 & 기대효과
- **난이도**: ★★★☆☆ (뉴스-주가 매칭 로직 + 통계 계산, 대부분 기존 데이터 활용)
- **기대효과**: 뉴스에 대한 감정적 반응 방지. "이 뉴스가 정말 주가에 영향을 주는가?"를 데이터로 판단. 뉴스 유형별 대응 전략 수립 근거.

## 아이디어 10: 커스텀 종목 스캐너 (조건 조합 검색)

### 개요
사용자가 **직접 조건을 조합**하여 두 프로젝트의 전체 데이터에서 종목을 검색하는 필터링 엔진. 텔레그램 명령어 또는 웹 UI로 실시간 조건 검색 가능.

### 핵심 가치
- **편의성**: "내가 원하는 조건의 종목"을 즉시 검색
- **분석 깊이**: 두 프로젝트의 모든 지표를 조합 가능
- **실시간성**: 장중 데이터 갱신 시 조건 재스캔 + 알림

### 사용 예시

#### 텔레그램 명령어
```
👤 /scan 적극매수 AND 외국인순매수>=3일 AND 테마순위<=5
🤖 조건 일치 종목 2건:
   1. 삼성SDI - 적극매수 | 외국인 4일 순매수 | 2차전지(#1)
   2. SK하이닉스 - 적극매수 | 외국인 5일 순매수 | AI반도체(#2)

👤 /scan MA정배열 AND 거래대금TOP30 AND 공매도<3%
🤖 조건 일치 종목 5건:
   1. 현대차 - MA정배열 | 거래대금 #8 | 공매도 1.2%
   ...

👤 /scan 어제매도→오늘매수 AND 거래량급증
🤖 조건 일치 종목 1건:
   1. 카카오 - 어제 매도→오늘 매수 전환 | 거래량 전일 대비 285%
```

### 검색 가능 조건 (필터 카탈로그)

| 카테고리 | 조건 | 데이터 출처 |
|----------|------|-----------|
